# JavaScript Coding Exercises

Welcome to the JavaScript Coding Exercises collection! These exercises are designed to help you master JavaScript fundamentals, DOM manipulation, asynchronous programming, and modern ES6+ features.

## Structure

- **Beginner**: Basic syntax, variables, functions, and simple logic
- **Intermediate**: Arrays, objects, DOM manipulation, and event handling
- **Advanced**: Async/await, promises, closures, and design patterns

## How to Use

1. Navigate to the exercise file you want to work on
2. Read the problem description carefully
3. Write your solution in the designated area
4. Run the tests to verify your solution
5. Check the solutions folder if you get stuck

## Running Exercises

Open the HTML file in your browser or run with Node.js:
```bash
node exercises/beginner/exercise-name.js
```

## Topics Covered

- Variables and Data Types
- Functions and Scope
- Arrays and Objects
- DOM Manipulation
- Event Handling
- Asynchronous JavaScript
