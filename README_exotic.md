# Hello World Collection in Exotic Programming Languages

This collection includes "Hello, World!" programs written in various esoteric and exotic programming languages.

## Languages Included

### Esoteric Languages
- **Brainfuck** (`.brainfuck`) - A minimalist language with only 8 commands
- **Whitespace** (`.whitespace`) - Uses only whitespace characters (spaces, tabs, newlines)
- **Ook!** (`.ook`) - Based on Brainfuck but using only "Ook.", "Ook?", and "Ook!"
- **Malbolge** (`.malbolge`) - Designed to be the hardest programming language ever created
- **Befunge** (`.befunge`) - A two-dimensional stack-based language

### Themed Languages
- **INTERCAL** (`.intercal`) - Created as a parody language with bizarre syntax
- **Piet** (`.piet`) - Programs are abstract art images (text description provided)
- **COW** (`.cow`) - Uses only variations of "moo"
- **LOLCODE** (`.lolcode`) - Based on lolcat memes
- **Chef** (`.chef`) - Programs look like cooking recipes

### Creative Languages
- **Rockstar** (`.rockstar`) - Reads like rock song lyrics
- **ArnoldC** (`.arnoldc`) - Uses Arnold Schwarzenegger movie quotes
- **Zombie** (`.zombie`) - Zombie-themed programming language
- **Taxi** (`.taxi`) - Programs describe taxi routes
- **Grass** (`.grass`) - Uses only 'w' and 'W' characters
- **Unlambda** (`.unlambda`) - A functional language based on combinatory logic

## Running the Programs

Note: Most of these languages require specific interpreters to run. These files are primarily for demonstration and educational purposes.
