# Hello Scripts

This directory contains simple "Hello, world!" scripts in multiple languages:

- `hello.sh` - Bash
- `hello.py` - Python 3
- `hello.js` - Node.js
- `hello.rb` - Ruby
- `hello.awk` - awk
- `hello.pl` - Perl
- `run-all.sh` - Run all available scripts

Make scripts executable: `chmod +x hello-scripts/*`
Run: `./hello-scripts/hello.sh` etc.

Run everything: `./hello-scripts/run-all.sh`
