# Hello World Scripts Collection

This directory contains various "Hello, World!" implementations demonstrating different programming concepts and languages.

## JavaScript Scripts

### 1. **hello-console.js**
Simple console output
```bash
node hello-console.js
```

### 2. **hello-node.js**
Node.js with system information
```bash
node hello-node.js
# or
./hello-node.js
```

### 3. **hello-dom.js**
DOM manipulation (requires browser environment)
- Include in an HTML file with `<script src="hello-dom.js"></script>`

### 4. **hello-async.js**
Async/await demonstration
```bash
node hello-async.js
```

### 5. **hello-class.js**
Object-oriented approach with classes
```bash
node hello-class.js
```

### 6. **hello-server.js**
HTTP server that serves a hello world page
```bash
node hello-server.js
# Then visit http://localhost:3000
```

## HTML

### 7. **hello.html**
Standalone HTML page with styling and animation
- Open directly in a browser
- Features gradient background and fade-in animation

## Python

### 8. **hello-python.py**
Python script with multi-language greetings
```bash
python3 hello-python.py
# or
./hello-python.py
```

## Bash

### 9. **hello-bash.sh**
Bash script with colored output and system info
```bash
bash hello-bash.sh
# or
./hello-bash.sh
```

## Quick Test All Scripts

```bash
# JavaScript
node hello-console.js
node hello-node.js
node hello-async.js
node hello-class.js

# Python
python3 hello-python.py

# Bash
bash hello-bash.sh

# Server (runs in background)
node hello-server.js &
```

## Features Demonstrated

- **Basic Output**: Simple console logging
- **Async Programming**: Promises and async/await
- **OOP**: Classes and methods
- **DOM Manipulation**: Browser JavaScript
- **HTTP Server**: Basic web server
- **Multi-language**: Greetings in 6 languages
- **System Info**: Platform and environment details
- **Styling**: HTML/CSS animations and gradients
- **Shell Scripting**: Colored terminal output

---

All scripts are tested and working in the sandbox environment (Amazon Linux 2023, Node.js 22).
