# Hello World Scripts

This folder contains simple "Hello, world!" scripts in multiple languages.

- `hello/bash/hello.sh` — Bash
- `hello/python/hello.py` — Python 3
- `hello/node/hello.js` — Node.js

Each script is executable. Run them directly, for example:

```
./hello/bash/hello.sh
./hello/python/hello.py
./hello/node/hello.js
```
