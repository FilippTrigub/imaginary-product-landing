# Hello World - NovaSphere

Welcome to the NovaSphere Hello World collection!

## Files Created

### 1. hello.html
A simple, styled "Hello World" HTML page with NovaSphere branding.

### 2. hello.js
JavaScript file with hello world functions and console messages.

### 3. hello.css
CSS styles for hello world elements with animations.

### 4. hello-interactive.html
An interactive hello world page where users can enter their name and receive a personalized greeting.

## Usage

### View the HTML files
Open any of the HTML files in a web browser:
- `hello.html` - Simple static greeting
- `hello-interactive.html` - Interactive greeting with input

### Use the JavaScript
```javascript
// Include hello.js in your project
const { sayHello } = require('./hello.js');

console.log(sayHello('Developer'));
// Output: Hello Developer! Welcome to NovaSphere.
```

### Apply the CSS
```html
<!-- Include hello.css in your HTML -->
<link rel="stylesheet" href="hello.css">

<!-- Use the classes -->
<div class="hello-world">Hello World</div>
<div class="hello-world hello-world-animated">Animated Hello</div>
```
