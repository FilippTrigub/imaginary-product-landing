# Hello World - NovaSphere Project

Welcome to the NovaSphere Hello World files!

## Files Created

### 1. hello.html
A standalone HTML page with a beautiful "Hello World" message featuring:
- Gradient background
- Animated text
- Modern glassmorphism design
- Responsive layout

### 2. hello.js
JavaScript file with Hello World functionality:
- Console logging
- Dynamic message display
- Browser notification
- Modular exports for Node.js compatibility

### 3. hello.css
Stylesheet with Hello World styling:
- Gradient backgrounds
- Smooth animations
- Responsive design
- Modern typography

## Usage

### View HTML Page
Open `hello.html` in your browser to see the animated Hello World page.

### Use JavaScript
```javascript
// In browser
<script src="hello.js"></script>

// In Node.js
const { sayHello, displayHelloMessage } = require('./hello.js');
console.log(sayHello('Developer'));
```

### Apply CSS Styles
```html
<link rel="stylesheet" href="hello.css">
<div class="hello-world">
    <div class="greeting-box">
        <h1>Hello World!</h1>
        <p>Welcome to NovaSphere</p>
    </div>
</div>
```
