# Hello World Scripts Collection

This directory contains various "Hello World" scripts demonstrating different JavaScript approaches for the NovaSphere project.

## Available Scripts

### 1. **hello-console.js**
Simple console-based hello world that logs messages to the browser console.

**Usage:**
```html
<script src="hello-console.js"></script>
```

### 2. **hello-dom.js**
Creates a styled popup overlay with "Hello World" that appears on page load and fades out after 3 seconds.

**Usage:**
```html
<script src="hello-dom.js"></script>
```

### 3. **hello-alert.js**
Shows a browser alert with "Hello World" message when the page loads.

**Usage:**
```html
<script src="hello-alert.js"></script>
```

### 4. **hello-animated.js**
Creates an animated typing effect in the top-right corner with a blinking cursor.

**Usage:**
```html
<script src="hello-animated.js"></script>
```

### 5. **hello-interactive.js**
Adds an interactive button at the bottom of the page that displays "Hello World" when clicked.

**Usage:**
```html
<script src="hello-interactive.js"></script>
```

### 6. **hello.html**
Standalone HTML page with a beautiful animated "Hello World" display, including:
- Gradient background
- Floating particles animation
- Smooth fade-in effects
- Interactive buttons
- Console messages

**Usage:**
Open directly in browser: `hello.html`

### 7. **hello-node.js**
Node.js command-line script that displays system information and colorful terminal output.

**Usage:**
```bash
node hello-node.js
# or make it executable
chmod +x hello-node.js
./hello-node.js
```

## Integration Examples

### Add to existing HTML page:
```html
<!DOCTYPE html>
<html>
<head>
    <title>My Page</title>
</head>
<body>
    <h1>My Content</h1>
    
    <!-- Add any hello world script -->
    <script src="hello-interactive.js"></script>
</body>
</html>
```

### Run in Node.js:
```bash
node hello-node.js
```

### Test in browser console:
```javascript
// Copy and paste content from hello-console.js
console.log('Hello World!');
```
