## 🚀 À Propos

**NovaSphere v2.0** est un site web vitrine moderne présentant un appareil révolutionnaire d'informatique personnelle. Le projet met en avant une interface holographique innovante, un traitement quantique et des fonctionnalités de sécurité neuronale de pointe.

Ce site web est conçu pour présenter les capacités futuristes de l'appareil NovaSphere avec une interface utilisateur élégante et responsive, des animations fluides et une expérience utilisateur optimale.

## 📦 Installation

### Prérequis

- Un navigateur web moderne (Chrome, Firefox, Safari, Edge)
- Un serveur web local (optionnel, pour le développement)

### Étapes d'Installation

1. **Cloner ou télécharger le projet**
   ```bash
   git clone <url-du-repo>
   cd /vercel/sandbox
   ```

2. **Ouvrir directement dans le navigateur**
   ```bash
   # Ouvrir index.html dans votre navigateur
   open index.html
   ```

3. **Ou utiliser un serveur local (recommandé)**
   ```bash
   # Avec Python 3
   python3 -m http.server 8000
   
   # Avec Node.js (http-server)
   npx http-server -p 8000
   
   # Avec PHP
   php -S localhost:8000
   ```

4. **Accéder au site**
   ```
   http://localhost:8000
   ```

## 🎨 Personnalisation

### Modifier les Couleurs

Les couleurs principales sont définies dans `styles.css`. Recherchez les dégradés :

```css
/* Dégradé principal */
background: linear-gradient(135deg, #6e8efb, #a777e3);

/* Couleur d'accent */
color: #6e8efb;
```

### Modifier les Polices

La police actuelle est Montserrat. Pour la changer, modifiez l'import dans les fichiers HTML :

```html
<link href="https://fonts.googleapis.com/css2?family=VotrePolice:wght@300;400;600;700&display=swap" rel="stylesheet">
```

Et dans `styles.css` :

```css
body {
    font-family: 'VotrePolice', sans-serif;
}
```

### Ajouter de Nouvelles Sections

1. Créez la structure HTML dans `index.html`
2. Ajoutez les styles correspondants dans `styles.css`
3. Si nécessaire, ajoutez l'interactivité dans `script.js`

## 🐛 Débogage

### Problèmes Courants

**Les images ne s'affichent pas**
- Vérifiez que les fichiers SVG sont présents dans le dossier `images/`
- Vérifiez les chemins relatifs dans les fichiers HTML

**Les styles ne s'appliquent pas**
- Vérifiez que `styles.css` est correctement lié
- Videz le cache du navigateur (Ctrl+F5 ou Cmd+Shift+R)

**Le JavaScript ne fonctionne pas**
- Ouvrez la console du navigateur (F12)
- Vérifiez les erreurs JavaScript
- Assurez-vous que `script.js` est chargé en fin de body
