# Hello World Scripts Collection

This directory contains multiple "Hello World" implementations in various languages and formats.

## Available Scripts

### 1. **hello.html** - Interactive Web Page
A standalone HTML page with styled "Hello World" message.
- Features: Gradient background, animations, responsive design
- Usage: Open directly in a web browser
```bash
# Open in browser (example)
open hello.html
# or
xdg-open hello.html
```

### 2. **hello.js** - Browser JavaScript
JavaScript for DOM manipulation in a web page.
- Features: Console output and dynamic DOM element creation
- Usage: Include in an HTML page
```html
<script src="hello.js"></script>
```

### 3. **hello-node.js** - Node.js Script
Colorful console output with ASCII art and multilingual greetings.
- Features: ANSI colors, multiple languages, ASCII art
- Usage:
```bash
node hello-node.js
# or
./hello-node.js
```

### 4. **hello-server.js** - HTTP Server
A simple Node.js HTTP server serving "Hello World" HTML.
- Features: Full HTTP server with styled response
- Usage:
```bash
node hello-server.js
# Then visit: http://localhost:3000
```

### 5. **hello.py** - Python Script
Python implementation with colors and multilingual support.
- Features: ANSI colors, multiple languages, ASCII art
- Usage:
```bash
python3 hello.py
# or
./hello.py
```

### 6. **hello.sh** - Bash Script
Shell script with colored output and greetings.
- Features: ANSI colors, multiple languages, ASCII art
- Usage:
```bash
bash hello.sh
# or
./hello.sh
```

## Quick Test All Scripts

```bash
# Test Node.js version
node hello-node.js

# Test Python version
python3 hello.py

# Test Bash version
bash hello.sh

# Start HTTP server (Ctrl+C to stop)
node hello-server.js
```
