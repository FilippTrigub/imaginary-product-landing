# Hello World Scripts

This folder contains small "Hello, World!" example scripts in several languages.

Available scripts:

- `hello.sh` - Bash
- `hello.py` - Python 3
- `hello.js` - Node.js
- `hello.rb` - Ruby
- `hello.ps1` - PowerShell (Windows)

Run examples:

```bash
# Bash
./hello-world-scripts/hello.sh

# Python
python3 hello-world-scripts/hello.py

# Node
node hello-world-scripts/hello.js

# Ruby
ruby hello-world-scripts/hello.rb
```

On Unix-like systems make sure the scripts are executable: `chmod +x hello-world-scripts/*`.
