# Hello World Scripts Collection

This directory contains "Hello, World!" programs in multiple programming languages.

## Available Scripts

### 1. JavaScript (Node.js) - `hello.js`
```bash
node hello.js
# or
chmod +x hello.js && ./hello.js
```

### 2. Python - `hello.py`
```bash
python3 hello.py
# or
chmod +x hello.py && ./hello.py
```

### 3. Bash Shell - `hello.sh`
```bash
bash hello.sh
# or
chmod +x hello.sh && ./hello.sh
```

### 4. HTML - `hello.html`
Open in a web browser:
```bash
# Or open directly in your browser
firefox hello.html
# or
chrome hello.html
```

### 5. C - `hello.c`
```bash
gcc hello.c -o hello_c
./hello_c
```

### 6. Java - `hello.java`
```bash
javac hello.java
java hello
```

## Quick Test All Scripts

Run the test script to execute all available hello world programs:
```bash
bash run_all.sh
```

## Purpose

These scripts demonstrate:
- Basic syntax of each language
- Output formatting
- Timestamp generation
- Executable script headers (shebang)
- Simple styling (for HTML)

Each script outputs "Hello, World!" along with additional formatted information.
