Hello World Scripts
===================

Small collection of "hello world" examples in several languages.

Files:
- `hello.sh` - POSIX shell
- `hello.py` - Python 3
- `hello.js` - Node.js
- `hello.rb` - Ruby (if installed)
- `hello.go` - Go (if installed)
- `hello.ps1` - PowerShell script

To run (examples):
- `bash hello.sh`
- `python3 hello.py`
- `node hello.js`

All scripts in this folder are marked executable.
