# Hello World Scripts

This directory contains simple "Hello World" programs in different programming languages. These scripts demonstrate basic programming concepts and can be used for testing environments or as learning examples.

## Available Scripts

### JavaScript (`hello-world.js`)
A Node.js script that demonstrates:
- Basic hello world output
- Current time display
- Multilingual greetings
- Modern JavaScript features (template literals, object destructuring)

**Run with:**
```bash
node hello-world.js
```

### Python (`hello-world.py`)
A Python 3 script that demonstrates:
- Basic hello world output
- Current time using datetime module
- Multilingual greetings
- System information display
- Python docstrings and functions

**Run with:**
```bash
python3 hello-world.py
```

### Bash (`hello-world.sh`)
A shell script that demonstrates:
- Basic hello world output
- Current time using date command
- Multilingual greetings
- System information using shell commands
- Bash functions and variable usage

**Run with:**
```bash
./hello-world.sh
```

## Features

All scripts include:
- Simple "Hello, World!" message
- Current time display
- Multilingual greetings (English, Spanish, French, German, Japanese)
- System/environment information
- Consistent output formatting

## Usage

These scripts can be used for:
- Testing programming environments
- Learning basic syntax of different languages
- Demonstrating cross-language concepts
- Quick environment verification
