# NovaSphere v2.0

A modern, responsive landing page for NovaSphere - an imaginary next-generation personal computing device featuring holographic interfaces, quantum processing, and revolutionary battery technology.

## Overview

NovaSphere v2.0 showcases a futuristic computing platform through a sleek, professional landing page. This project demonstrates modern web design principles with clean HTML, CSS, and JavaScript.

## Features

- **Responsive Design**: Fully responsive layout that adapts to all screen sizes
- **Modern UI/UX**: Clean, professional design with smooth animations
- **Multiple Sections**:
  - Hero section with product introduction
  - Key features showcase
  - Pricing plans (Standard, Pro, Enterprise)
  - Product vision and roadmap
  - About section
  - Contact form
- **Navigation**: Smooth scrolling navigation between sections
- **Custom Typography**: Uses Montserrat font family from Google Fonts

## Getting Started

### Prerequisites

No build tools or dependencies required. This is a static website that runs directly in any modern web browser.

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd <repository-name>
```

2. Open the project:
```bash
# Open index.html in your browser
open index.html

# Or use a local server (recommended)
python -m http.server 8000
# Then visit http://localhost:8000
```

### Using with a Local Server

For the best experience, serve the files using a local web server:

**Python 3:**
```bash
python -m http.server 8000
```

**Node.js (with http-server):**
```bash
npx http-server -p 8000
```

**PHP:**
```bash
php -S localhost:8000
```

Then navigate to `http://localhost:8000` in your browser.
