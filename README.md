# NovaSphere v2 Landing Page

NovaSphere v2 is a concept landing page showcasing "the future of personal computing" with a holographic interface, quantum processing, and neural security. This repository contains all assets for the marketing site, including markup, styles, scripts, and media.

## Features

- **Hero section** introducing the NovaSphere device with call-to-action buttons
- **Features grid** highlighting holographic interface, quantum processing, neural security, and infinite battery
- **Pricing plans** with Standard, Pro, and Enterprise tiers
- **Product vision** narrative with statistics, long-term roadmap, and imagery
- **About section** describing the team vision
- **Contact form** for collecting user inquiries
- **Footer** with navigation, contact details, and legal links

## Project Structure


## Getting Started

1. Clone or download this repository.
2. Open `index.html` in any modern desktop or mobile browser.
3. Ensure the `images/` directory remains relative to the HTML files so assets load correctly.

> The page is fully static and requires no build process or external dependencies beyond the hosted Google Font (Montserrat).

## Customization

- Update copy in `index.html` to reflect your product messaging.
- Modify color palette and typography in `styles.css` to match your brand identity.
- Replace `images/logo.svg` and `images/product.svg` with your own assets, keeping the same filenames or updating the references in HTML.
- Enhance interactivity by adding logic in `script.js` (currently empty).

## Accessibility & Performance Notes

- Semantic HTML structure with clear headings and section IDs for navigation
- Form controls include associated labels
- Responsive layout handled via CSS flexbox
- Minimal external resources to keep load times fast
