## 🌟 Descripción General

NovaSphere v2.0 es una página de inicio conceptual que demuestra principios de diseño web de vanguardia y técnicas modernas de HTML/CSS/JavaScript. El sitio presenta un dispositivo informático holográfico ficticio con una experiencia de usuario inmersiva y profesional.

## 🚀 Inicio Rápido

### Requisitos Previos
- Un navegador web moderno (Chrome, Firefox, Safari, Edge)
- Opcional: Servidor web local para desarrollo

### Instalación

1. **Clonar o descargar el repositorio**
   ```bash
   git clone <url-del-repositorio>
   cd sandbox
   ```

2. **Abrir en el navegador**
   - Simplemente abre `index.html` en tu navegador web
   - O usa un servidor de desarrollo local:
   
   ```bash
   # Con Python 3
   python3 -m http.server 8000
   
   # Con Node.js (http-server)
   npx http-server -p 8000
   
   # Con PHP
   php -S localhost:8000
   ```

3. **Acceder al sitio**
   - Abre `http://localhost:8000` en tu navegador

## 🎨 Diseño y Tecnología

### Sistema de Diseño
- **Fuente**: Montserrat (Google Fonts) - pesos 300, 400, 600, 700
- **Paleta de Colores**: 
  - Primario: Degradados azules modernos
  - Fondo: Gris claro (#f8f9fa) con superposiciones de degradados
  - Texto: Gris oscuro (#333) con ratios de contraste accesibles
- **Diseño**: CSS Grid y Flexbox para diseños responsivos
- **Animaciones**: Transiciones CSS suaves y efectos al pasar el cursor

### Tecnologías Utilizadas
- **HTML5** - Marcado semántico
- **CSS3** - Estilización moderna con Grid, Flexbox, animaciones
- **JavaScript Vanilla** - Funcionalidades interactivas sin dependencias
- **Gráficos SVG** - Imágenes vectoriales escalables para logo y producto

## 🛠️ Personalización

### Actualización del Contenido
1. **Contenido de Texto**: Edita directamente los archivos HTML
2. **Estilización**: Modifica `styles.css` para colores, fuentes, espaciado
3. **Imágenes**: Reemplaza los archivos SVG en el directorio `/images/`
4. **Miembros del Equipo**: Actualiza la cuadrícula de equipo en `team.html`

### Agregar Nuevas Secciones
1. Crea una nueva sección en HTML con las etiquetas semánticas apropiadas
2. Agrega los estilos correspondientes en `styles.css`
3. Actualiza los enlaces de navegación si es necesario
4. Agrega interacciones JavaScript en `script.js` si se requiere
