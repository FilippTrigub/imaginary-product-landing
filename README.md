# NovaSphere Landing Page

NovaSphere v2 imagines the next era of personal computing with holographic interfaces, quantum-level performance, and adaptive AI. This repo holds the marketing site that introduces the concept device and spotlights its headline capabilities.

## What's Inside
- Responsive one-page experience in `index.html` with hero, feature, pricing, and vision sections.
- Shared layout bits extracted to `header.html`, plus a dedicated `team.html` feature page.
- Modern styling in `styles.css` and interaction flourishes in `script.js`.
- Vector illustrations and logos in `images/`.

## Quick Start
1. Clone or download this repository.
2. Open `index.html` directly in your browser, or serve the folder locally for hot reload and relative paths that just work:
   ```bash
   npx serve .
   ```
3. Peek at `team.html` for the companion section that highlights the NovaSphere crew.

## Customize the Page
- Swap copy inside the HTML files to tailor messaging.
- Replace assets in `images/` with your own SVGs or PNGs (keep the filenames if you do not want to touch markup).
- Adjust colors, gradients, and spacing in `styles.css`; component blocks are grouped and annotated for quicker scanning.
- Extend interactivity or hook up analytics inside `script.js`.
