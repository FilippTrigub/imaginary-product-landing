# NovaSphere v2

## Project Overview
NovaSphere v2 is a web project that showcases a revolutionary personal computing device with a holographic interface. The project consists of several HTML pages that provide information about the product, its features, pricing, and the team behind it.

## HTML Files Overview

### 1. `header.html`
This file contains the enhanced header navigation for the website. Key features include:
- **Sticky Header**: The header remains at the top of the viewport while scrolling.
- **Dropdown Menus**: Navigation links for products and company sections with subpage links.
- **Mobile Responsiveness**: A toggle button for mobile menu display.
- **Breadcrumb Navigation**: Indicates the current page location.
- **Styling**: Uses a gradient background and hover effects for links.

### 2. `index.html`
The main landing page for NovaSphere v2. It includes:
- **Hero Section**: Introduces the product with a call-to-action to watch a demo or pre-order.
- **Features Section**: Highlights key features such as holographic interface, quantum processing, and neural security.
- **Pricing Plans**: Displays different pricing tiers for the product.
- **About Section**: Provides background information about NovaSphere.
- **Contact Form**: Allows users to get in touch with the company.

### 3. `team.html`
This page introduces the team behind NovaSphere. It features:
- **Team Members**: Profiles of key team members including their roles and bios.
- **Grid Layout**: A responsive grid layout for displaying team member information.

## Conclusion
The NovaSphere v2 project aims to provide a modern and engaging user experience while showcasing innovative technology in personal computing. The HTML structure is designed to be responsive and user-friendly, ensuring accessibility across devices.
