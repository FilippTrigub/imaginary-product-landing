## Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Open `index.html` in a web browser or serve with a local server:
   ```bash
   python -m http.server 8000
   # or
   npx http-server
   ```

3. Visit `http://localhost:8000` to view the landing page
