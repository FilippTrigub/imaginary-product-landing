## Aperçu

NovaSphere v2.0 introduit une façon révolutionnaire d'interagir avec la technologie. Fondée en 2024, NovaSphere réunit ingénieurs, designers et penseurs pour créer des outils qui étendent la créativité humaine. Nos produits se concentrent sur des interactions agréables et significatives qui s'adaptent des créateurs individuels aux grandes organisations.

Ce dépôt contient la page d'accueil et le site web marketing de NovaSphere, présentant notre plateforme informatique holographique innovante.

## Fonctionnalités principales

### 🌐 Interface holographique
Interagissez avec du contenu numérique en trois dimensions, donnant vie à votre travail et vos divertissements avec des hologrammes intuitifs qui semblent naturels.

### ⚡ Traitement quantique
Notre puce quantique propriétaire offre des performances sans précédent tout en consommant une énergie minimale, garantissant des interactions fluides et réactives.

### 🔒 Sécurité neuronale
Authentification biométrique avancée qui apprend et s'adapte à vos schémas uniques, fournissant une IA adaptative qui personnalise votre espace de travail.

### 🔋 Batterie infinie
Technologie révolutionnaire de récupération d'énergie qui maintient votre appareil alimenté jusqu'à 2 semaines, avec une alimentation durable qui réduit le gaspillage.

### 🔊 Audio spatial
Technologie audio spatiale qui positionne le son là où il doit être, créant une expérience audio immersive.

## Démarrage

Il s'agit d'un site web statique construit avec HTML, CSS et JavaScript. Aucun processus de compilation n'est requis.

### Prérequis

- Un navigateur web moderne (Chrome, Firefox, Safari ou Edge)
- Un serveur web local (optionnel, pour le développement)

### Installation

1. Clonez le dépôt :
```bash
git clone https://github.com/novasphere/novasphere-landing.git
cd novasphere-landing
```

2. Ouvrez le site web :
   - **Option A** : Ouvrez simplement `index.html` dans votre navigateur web
   - **Option B** : Utilisez un serveur web local :
   ```bash
   # Avec Python 3
   python -m http.server 8000

   # Avec Node.js (avec http-server)
   npx http-server
   ```

3. Visitez `http://localhost:8000` dans votre navigateur
