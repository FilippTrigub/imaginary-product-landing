## 🛠️ Setup and Installation

This is a static website with no build process required. Simply clone the repository and open the files in your browser.

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)

### Installation
1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd novasphere-landing
   ```

2. Open `index.html` in your web browser to view the landing page.

## 🎯 Usage

- Navigate through the landing page to explore NovaSphere's features
- Visit the team page via the navigation menu
- All interactions are handled client-side with vanilla JavaScript
