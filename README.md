# NovaSphere v2 - The Future of Personal Computing

A modern, responsive landing page showcasing NovaSphere, a fictional cutting-edge computing product with holographic interfaces and quantum processing technology.

## 🚀 Project Overview

NovaSphere is a comprehensive marketing website built with vanilla HTML, CSS, and JavaScript that demonstrates modern web development practices without the need for frameworks or build tools. The site features smooth animations, responsive design, and an engaging user experience.

## ✨ Features

### 🎨 Design & UI
- **Responsive Design**: Mobile-first approach with breakpoints at 768px and 992px
- **Modern Styling**: CSS Grid, Flexbox, gradients, and smooth transitions
- **Interactive Animations**: Hover effects, scroll-triggered animations, and smooth scrolling
- **Typography**: Google Fonts (Montserrat) for professional appearance
- **Color Scheme**: Purple-blue gradient theme (#667eea to #764ba2)

### 📱 Pages & Components
- **Landing Page** (`index.html`): Complete product showcase with hero, features, pricing, and contact
- **Enhanced Header** (`header.html`): Advanced navigation with dropdowns and mobile menu
- **Team Page** (`team.html`): Professional team member profiles with bios

### 🛠️ Interactive Features
- Smooth scrolling navigation
- Contact form with success messaging
- Mobile hamburger menu with outside-click detection
- On-scroll animations for feature cards
- Floating product image animation
- Pricing card hover effects

## 🚀 Getting Started

### Prerequisites
- A modern web browser
- Local web server (optional but recommended)

### Installation

1. **Clone the repository:**
   ```bash
   git clone <repository-url>
   cd novasphere
   ```

2. **Open locally:**
   - Option A: Open `index.html` directly in your browser
   - Option B: Use a local server (recommended):
     ```bash
     # Python 3
     python -m http.server 8000

     # Node.js (with http-server)
     npx http-server

     # PHP
     php -S localhost:8000
     ```

3. **Navigate to:** `http://localhost:8000`

## 🎨 Design System

- **Background**: `#f8f9fa` (Light gray)

### Components
- **Buttons**: Primary (.btn-primary) and Secondary (.btn-secondary) styles

- **Grid**: Responsive layouts with proper spacing

## 📱 Responsive Breakpoints

- **Desktop**: Default layout (1200px max-width containers)
- **Tablet**: 992px and below (single column layouts, scaled elements)
- **Mobile**: 768px and below (mobile menu, collapsed layouts)
