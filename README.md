# NovaSphere v2 - Landing Page

A modern, responsive landing page for NovaSphere, a futuristic holographic computing device. This project showcases an imaginary product with cutting-edge features like holographic interfaces, quantum processing, and neural security.

## Overview

NovaSphere v2 represents the future of personal computing through an innovative landing page design featuring:

- Clean, modern interface with smooth animations
- Fully responsive layout for all device sizes
- Comprehensive product showcase with features, pricing, and company information
- Interactive contact form
- Professional typography using Montserrat font family

## Getting Started

### Prerequisites

No build tools or dependencies required. This is a static HTML/CSS/JS project.

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd novasphere
```

2. Open in a browser:
```bash
# Using Python 3
python3 -m http.server 8000

# Or simply open index.html in your browser
open index.html
```

3. Navigate to `http://localhost:8000` (if using a local server)

## Development

The project uses a simple structure with no build process:

- Edit HTML files directly for content changes
- Modify `styles.css` for styling updates
- Update `script.js` for interactive features

### Customization

To customize the landing page:

1. **Colors** - Update CSS variables in `styles.css`
2. **Content** - Modify text directly in `index.html`
3. **Images** - Replace files in the `images/` directory
4. **Typography** - Change font imports in the `<head>` section
