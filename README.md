## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- Optional: Local web server for development

### Installation

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd /vercel/sandbox
   ```

2. **Open in browser**
   
   Simply open `index.html` in your web browser:
   ```bash
   # Using Python 3
   python3 -m http.server 8000
   
   # Using Node.js
   npx http-server -p 8000
   
   # Or just open the file directly
   open index.html  # macOS
   xdg-open index.html  # Linux
   start index.html  # Windows
   ```

3. **View the site**
   
   Navigate to `http://localhost:8000` in your browser

## 🔧 Customization

### Updating Content
1. Edit text content directly in `index.html` and `team.html`
2. Modify styles in `styles.css`
3. Update interactive behavior in `script.js`

### Changing Colors
Update CSS variables or color values in `styles.css`:
```css
/* Example: Update primary button color */
.btn-primary {
    background: linear-gradient(135deg, #your-color-1, #your-color-2);
}
```

### Adding New Sections
Follow the existing HTML structure and CSS class naming conventions for consistency.
