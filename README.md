# NovaSphere - The Future of Personal Computing

This repository contains the landing page for NovaSphere, an imaginary product that features holographic interfaces and advanced computing technologies.

## Hello World Scripts

Inside the `scripts` directory, you can find portable "Hello, World!" examples:

- `hello.sh`: Bash script (`./scripts/hello.sh`)
- `hello.py`: Python script (`python scripts/hello.py`)
- `hello.js`: Node.js script (`node scripts/hello.js`)
