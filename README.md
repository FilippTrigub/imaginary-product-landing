# NovaSphere - The Future of Personal Computing

This repository contains the source code for the NovaSphere landing page, showcasing an innovative, imaginary product that redefines personal computing through holographic interfaces and advanced technologies.

## Project Structure

- `index.html`: The main landing page.
- `header.html`: Contains the common header section used across pages.
- `team.html`: Information about the NovaSphere team.
- `styles.css`: Stylesheets for the website.
- `script.js`: JavaScript for interactive elements.
- `images/`: Directory containing product logos and other assets.

## How to View

To view the website, simply open the `index.html` file in your web browser.

```bash
# Using a web server (recommended for full functionality, though not strictly necessary for this static site)
# You can use a simple Python HTTP server:
python3 -m http.server 8000

# Then open your browser to http://localhost:8000
```
