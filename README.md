# NovaSphere v2 - Landing Page

A modern, responsive landing page for NovaSphere, showcasing revolutionary holographic computing technology.

## Features

- **Responsive Design** - Optimized for desktop, tablet, and mobile devices
- **Modern UI/UX** - Clean interface with smooth animations and transitions
- **Multiple Pages** - Home page, team page, and enhanced header navigation
- **Interactive Elements** - Dropdown menus, smooth scrolling, and form handling
- **Professional Styling** - Gradient effects, card layouts, and modern typography

## Project Structure

```
/vercel/sandbox/
├── header.html          # Enhanced header with navigation demo
├── team.html            # Team members page
├── styles.css           # Main stylesheet
├── script.js            # Interactive functionality
├── main.js              # Additional JavaScript
├── images/
│   ├── logo.svg         # NovaSphere logo
│   └── product.svg      # Product illustration
└── README.md            # This file
```

## Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Modern styling with flexbox and grid
- **JavaScript** - Vanilla JS for interactivity
- **Google Fonts** - Montserrat font family

## Getting Started

1. Clone or download this repository
2. Open `header.html` or `team.html` in your web browser
3. No build process or dependencies required

## Pages

- **header.html** - Enhanced navigation system with dropdown menus and breadcrumbs
- **team.html** - Meet the team behind NovaSphere

## Customization

- Modify `styles.css` to change colors, fonts, and layout
- Update `script.js` to add or modify interactive features
- Edit HTML files to change content and structure

- Firefox (latest)
