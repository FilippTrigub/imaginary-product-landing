# NovaSphere - The Future of Personal Computing

This repository contains the landing page for NovaSphere, an imaginary product that features holographic interfaces and advanced computing technologies.

## Hello World scripts

- `hello_world_bash.sh`: prints "Hello, World!" from Bash. Run with `./hello_world_bash.sh`.
- `hello_world_python.py`: prints "Hello, World!" from Python. Run with `python hello_world_python.py`.
- `hello_world_node.js`: prints "Hello, World!" from Node.js. Run with `node hello_world_node.js`.
