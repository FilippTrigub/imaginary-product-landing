## 🎯 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- Basic understanding of HTML/CSS/JavaScript for modifications

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/novasphere-landing.git
   cd novasphere-landing
   ```

2. **Open in browser**
   - Simply open `index.html` in your preferred web browser
   - Or use a local development server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js (if you have it installed)
   npx serve .
   ```

3. **View the site**
   - Navigate to `http://localhost:8000` (if using a server)
   - Or directly open the HTML files in your browser

## 🔧 Customization

### Updating Content
1. **Product Information**: Edit the content in `index.html`
2. **Pricing**: Modify the pricing cards section
3. **Team Members**: Update `team.html` with your team info
4. **Colors**: Adjust CSS custom properties in `styles.css`
5. **Images**: Replace SVG files in the `images/` directory

### Adding New Sections
1. Add HTML structure to `index.html`
2. Create corresponding CSS in `styles.css`
3. Add any interactive features to `script.js`
4. Update navigation links as needed

## 🌟 Key Interactions

- **Smooth Scrolling**: Click navigation links for animated scrolling
- **Form Submission**: Contact form shows success message
- **Scroll Animations**: Feature cards animate into view on scroll
