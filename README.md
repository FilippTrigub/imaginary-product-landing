# NovaSphere v2.0 - Landing Page

A modern, responsive landing page for NovaSphere, a fictional revolutionary personal computing device featuring holographic interfaces and advanced technology.

## Overview

NovaSphere v2.0 is a conceptual product landing page showcasing futuristic computing technology with holographic displays, quantum processing, neural security, and innovative battery technology. This project demonstrates modern web design principles with clean aesthetics and user-friendly navigation.

## Features

- **Responsive Design**: Optimized for all device sizes and screen resolutions
- **Modern UI/UX**: Clean, professional interface with smooth animations
- **Multiple Sections**:
  - Hero section with product showcase
  - Key features overview
  - Pricing plans (Standard, Pro, Enterprise)
  - Product vision and roadmap
  - About section
  - Contact form
  - Comprehensive footer

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- A local web server (optional, for development)

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd novasphere
```

2. Open the project:
```bash
# Option 1: Open directly in browser
open index.html

# Option 2: Use a local server (recommended)
python3 -m http.server 8000
# Then navigate to http://localhost:8000
```

## Customization

### Colors
Main brand colors are defined in `styles.css` and can be customized:
- Primary: Various shades of blue and purple
- Accents: Gradients and modern color schemes

### Content
Update text content directly in the HTML files:
- Hero messaging in `index.html` lines 60-65
- Features in `index.html` lines 76-97
- Pricing in `index.html` lines 104-143

### Images
Replace SVG files in the `images/` directory:
- `logo.svg`: Company logo
- `product.svg`: Product visualization
