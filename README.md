# NovaSphere — Landing Page

A compact static website that showcases a fictional product called NovaSphere. The repo contains the HTML, CSS, JS, and assets used for the landing page. It's intended as an example of a small, responsive marketing site with simple interactivity.

**Quick Start**
- Preview in a browser: open `index.html` directly in any modern browser.
- Run a local static server (recommended):
  - Using `node` (needs Node.js 12+): `npx http-server -c-1` then open `http://localhost:8080`.
  - Using Python 3: `python -m http.server 8000` then open `http://localhost:8000`.

**Project Structure**
- `index.html` — Main landing page HTML.
- `header.html` — Reusable header include fragment.
- `team.html` — Team / about page.
- `styles.css` — Global styles and layout rules.
- `script.js` — JavaScript for interactive behavior.
- `images/` — Image assets used by the site.
- `README.md` — This file.

**Development Notes**
- The site is static and does not require a build step.
- Edit `styles.css` and `script.js` to change visuals and interactivity.
- If you add new assets to `images/`, reference them with relative paths in HTML/CSS.

**Testing**
- Manual testing is sufficient: open pages in multiple browsers and viewport sizes.
