# NovaSphere - The Future of Personal Computing

This repository contains the landing page for NovaSphere, an imaginary product that features holographic interfaces and advanced computing technologies.

## Hello World scripts

Simple "hello world" examples live in `scripts/`:

- Bash: `bash scripts/hello-world.sh`
- Node: `node scripts/hello-world.js`
- Python: `python3 scripts/hello-world.py`
- HTML: open `scripts/hello-world.html` in a browser
