# Sandbox Environment

This repository is a sandbox environment for executing code.

## Environment Details

*   **Operating System:** Amazon Linux 2023
*   **Package Manager:** dnf
*   **Runtimes:**
    *   Node.js 22
    *   Python 3
*   **Python Package Manager:** pip (must be installed before use)

## Project Context

This is a generic sandbox environment. You can create files and write code for any type of project.
