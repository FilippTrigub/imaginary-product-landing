# NovaSphere v2 - Landing Page

A modern, responsive landing page for NovaSphere, a fictional revolutionary personal computing device featuring holographic interfaces, quantum processing, and advanced AI capabilities.

## Project Overview

NovaSphere v2 is a multi-page static website showcasing an imaginary product with cutting-edge features. The site demonstrates modern web design principles with a clean, professional aesthetic and engaging user experience.

## Features

### Pages
- **index.html** - Main landing page with hero section, features, pricing, product vision, about section, and contact form
- **team.html** - Team page showcasing the brilliant minds behind NovaSphere
- **header.html** - Enhanced header demo with advanced navigation system

### Key Sections
- **Hero Section** - Eye-catching introduction with call-to-action buttons
- **Features Grid** - Four key product features with icons and descriptions
- **Pricing Plans** - Three-tier pricing structure (Standard, Pro, Enterprise)
- **Product & Vision** - Detailed product philosophy and 5-year roadmap
- **Team Showcase** - Meet the leadership team
- **Contact Form** - Get in touch with NovaSphere
- **Footer** - Comprehensive navigation and company information

### Technical Features
- Responsive design optimized for desktop, tablet, and mobile
- Custom CSS animations and hover effects
- Sticky header navigation with dropdowns
- Breadcrumb navigation system
- Mobile-responsive menu with toggle
- Smooth scrolling for anchor links
- ASCII art branding in HTML comments
- Modern gradient backgrounds and glassmorphism effects

## Technology Stack

- **HTML5** - Semantic markup structure
- **CSS3** - Custom styling with flexbox and grid layouts
- **JavaScript** - Interactive navigation and form handling
- **Google Fonts** - Montserrat font family
- **SVG Graphics** - Scalable logo and product images
