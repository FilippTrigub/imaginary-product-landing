## 📱 Responsive Design

The website is fully responsive and optimized for:

- **Desktop** (1200px+): Full layout with all features
- **Tablet** (768px - 1199px): Adapted grid layouts
- **Mobile** (320px - 767px): Stacked layout with hamburger menu

## 🔧 Customization Guide

### Colors & Branding
```css
/* Primary gradient colors */
--primary-gradient: linear-gradient(135deg, #6e8efb, #a777e3);
--secondary-color: #f8f9fa;
--text-dark: #333;
```

### Typography
```css
/* Font family */
font-family: 'Montserrat', sans-serif;
/* Available weights: 300, 400, 600, 700 */
```

### Layout Modifications
- Modify grid layouts in the CSS for different column arrangements
- Adjust container max-width for different page widths
- Update breakpoints for custom responsive behavior
