# NovaSphere v2 — The Future of Personal Computing

This repository contains the static landing site for NovaSphere v2, a fictional product showcasing a futuristic personal computing experience.

Key highlights
- Holographic Interface: 3D interaction for work and entertainment.
- Quantum Processing: experimental chip design for high performance and low energy.
- Neural Security: adaptive biometric protection.

Repository contents
- `index.html` — Main landing page.
- `team.html` — Team / about page.
- `header.html` — Enhanced header/navigation fragment.
- `styles.css` — Styles for the site.
- `script.js` — Frontend scripts and interactive behaviors.
- `images/` — Graphics and SVG assets used by the site.

Run locally
- Using Node: `npx http-server -c-1` (serves current folder at `http://localhost:8080`).
- Using Python 3: `python -m http.server 8000` and open `http://localhost:8000`.

Deployment
- This is a static site and can be deployed to any static host (Vercel, Netlify, GitHub Pages, S3).
