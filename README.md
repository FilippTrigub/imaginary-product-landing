# NovaSphere v2 - Landing Page

This repository contains the source code for the NovaSphere v2 landing page, a modern and visually appealing website for a fictional futuristic computing device.

## Overview

NovaSphere v2 is a concept for a next-generation personal computing device featuring a holographic interface, quantum processing, and neural security. This landing page is designed to showcase the product's features and generate interest through a clean and engaging user experience.

The landing page is a single-page design, divided into the following sections:

*   **Hero:** A captivating introduction with a call-to-action.
*   **Features:** A grid-based layout highlighting the key features of NovaSphere v2.
*   **Pricing:** A comparison of different pricing plans.
*   **Product & Vision:** A detailed look into the product's design and future goals.
*   **About:** A brief description of the company.
*   **Contact:** A simple and functional contact form.
*   **Footer:** with all the main links of the page.

## Features

*   **Responsive Design:** The layout adapts to different screen sizes, providing a seamless experience on desktops, tablets, and mobile devices.
*   **Smooth Scrolling:** Navigation links smoothly scroll to the corresponding sections of the page.
*   **Interactive Elements:** Subtle animations and hover effects enhance the user experience.
*   **Contact Form:** A functional contact form with basic client-side validation.

## Technologies Used

*   **HTML5:** The core markup language for the website's structure.
*   **CSS3:** For styling and creating a visually appealing layout.
*   **JavaScript:** For interactive features like smooth scrolling and form handling.
*   **Google Fonts:** For custom typography.

## How to Use

To view the landing page, simply open the `index.html` file in your web browser. No special setup or dependencies are required.
