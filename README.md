# NovaSphere - The Future of Personal Computing

This repository contains the landing page for NovaSphere, an imaginary product that features holographic interfaces and advanced computing technologies.

## Hello World Scripts

We've added several "hello world" scripts in different programming languages to demonstrate basic syntax:

### Files Included:

- `hello_world.html` - A simple HTML page with "Hello, World!"
- `hello_world.js` - A JavaScript script that outputs "Hello, World!" to the console
- `hello_world.py` - A Python script that prints "Hello, World!"
- `hello_world.sh` - A shell script that echoes "Hello, World!"

### How to Run:

- **HTML**: Open `hello_world.html` in any web browser
- **JavaScript**: Run `node hello_world.js` in your terminal
- **Python**: Run `python3 hello_world.py` in your terminal
- **Shell**: Run `chmod +x hello_world.sh && ./hello_world.sh` in your terminal

These examples demonstrate the minimal code required to output the classic "Hello, World!" greeting in each language.
