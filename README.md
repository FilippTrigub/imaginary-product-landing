# NovaSphere — The Future of Personal Computing

A small static landing site demonstrating a fictional product called "NovaSphere". This repository contains HTML, CSS, JavaScript and image assets that make up a simple marketing/landing page.

## Project structure

- index.html — The main landing page
- header.html — Reusable header fragment included by the main page
- team.html — Team / about page
- styles.css — Global styles for the site
- script.js — Client-side JavaScript for interactive behaviors
- images/ — SVG and raster image assets (logo.svg, product.svg)

## Viewing the site locally

This is a static site. To view it locally, open `index.html` in your browser (File → Open File or drag the file into the browser window).

Important: Do not run or start any development servers to view this project. Opening the HTML files directly in the browser is sufficient and recommended.

## Development notes

- This project uses plain HTML, CSS and vanilla JavaScript. Follow existing coding style when making edits.
- Keep assets inside the `images/` directory. Prefer SVG for logos and UI illustrations.
- Avoid introducing build tools, package managers, or server-side code unless explicitly requested.
