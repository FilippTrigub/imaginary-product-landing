# NovaSphere v2 Landing Page

Marketing site for the fictional NovaSphere v2 device, showcasing a holographic computing platform through a modern single-page experience plus supporting assets.

## Overview
- Responsive landing page built with semantic HTML, custom CSS, and lightweight JavaScript.
- Includes hero, feature, pricing, product vision, contact, and footer sections with cohesive branding.
- Ships with a secondary `team.html` page and shared header partial for future expansion.

## Features
- Hero layout with device imagery and dual call-to-action buttons.
- Feature grid highlighting holographic interface, quantum processing, neural security, and long-life power.
- Pricing cards with standout plan styling and callouts for pre-orders.
- Vision section with stats, emoji bullets, and contextual CTAs.
- Contact form markup ready for backend integration.

## Tech Stack
- HTML5 for structure
- CSS3 with flexbox/grid for layout and animations
- Vanilla JavaScript (`script.js`) placeholder for interactive enhancements
- Google Fonts (Montserrat) for typography

## Getting Started
1. Clone the repository: `git clone <repo-url>`
2. Open `index.html` directly in your browser **or** serve locally: `npx serve` and visit the provided URL.
3. Update asset references in `images/` or content sections as needed.

## Project Structure
```
.
├── index.html        # Main landing page
├── team.html         # Example secondary page
├── header.html       # Shared header snippet
├── styles.css        # Global styles and layout rules
├── script.js         # Placeholder for future interactions
└── images/           # SVGs and illustrations for branding
```

## Customization Tips
- Adjust copy, CTA targets, and pricing tiers in `index.html` to match your product messaging.
- Extend or replace imagery under `images/` to align with updated branding.
- Use `styles.css` to tweak color palette, spacing, or animation timing.
- Connect the contact form to your preferred backend or service provider before launch.
