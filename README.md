# NovaSphere v2.0

> The Future of Personal Computing

A modern, responsive landing page for NovaSphere - a conceptual next-generation personal computing device featuring holographic interfaces, quantum processing, and advanced AI capabilities.

## Overview

NovaSphere v2.0 is a fully responsive marketing website showcasing an imaginary cutting-edge computing platform. The site features a clean, modern design with smooth animations and an intuitive user experience.

## Features

- **Responsive Design**: Fully optimized for desktop, tablet, and mobile devices
- **Modern UI/UX**: Clean interface with smooth animations and transitions
- **Product Showcase**: Comprehensive features section highlighting key capabilities
- **Pricing Plans**: Three-tier pricing structure (Standard, Pro, Enterprise)
- **Product Vision Section**: Detailed product philosophy and 5-year roadmap
- **Contact Form**: Interactive contact form for user inquiries
- **Team Page**: Dedicated page showcasing the NovaSphere team
- **Consistent Branding**: Cohesive design system across all pages

## Technology Stack

- **HTML5**: Semantic markup with accessibility considerations
- **CSS3**: Modern styling with custom properties, flexbox, and grid
- **JavaScript**: Vanilla JS for interactive components
- **Google Fonts**: Montserrat font family for typography

## Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- Basic web server (optional, for local development)

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd novasphere
```

2. Open in browser:
   - Simply open `index.html` in your web browser, or
   - Use a local web server:

```bash
# Using Python 3
python3 -m http.server 8000

# Using Node.js (http-server)
npx http-server -p 8000
```

3. Navigate to `http://localhost:8000` in your browser

## Development

### File Descriptions

- **index.html**: Main landing page with all primary sections
- **team.html**: Team showcase page with member profiles
- **header.html**: Shared header component for consistent navigation
