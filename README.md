# NovaSphere - The Future of Personal Computing

This repository contains the landing page for NovaSphere, an imaginary product that features holographic interfaces and advanced computing technologies.

## Hello World scripts

Runnable examples live in `scripts/`:

- Node.js: `node scripts/hello.js`
- Bash: `bash scripts/hello.sh`
- Python: `python3 scripts/hello.py`
- Run all (bash + node + python): `bash scripts/run-hello.sh`
- HTML: open `scripts/hello.html` in a browser
