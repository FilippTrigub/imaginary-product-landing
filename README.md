# NovaSphere - The Future of Personal Computing

This repository contains the landing page for NovaSphere, an imaginary product that features holographic interfaces and advanced computing technologies.

## Hello World Scripts

Three minimal hello world scripts live in the project root for quick sanity checks:

- `hello_world.js` – run with `node hello_world.js`
- `hello_world.py` – run with `python hello_world.py`
- `hello_world.sh` – run with `./hello_world.sh`
