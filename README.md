# NovaSphere - The Future of Personal Computing

This repository contains the landing page for NovaSphere, an imaginary product that features holographic interfaces and advanced computing technologies.

## Getting Started

This is a simple npm app that serves a landing page.

### Installation

```bash
npm install
```

### Running the App

```bash
npm start
```

The app will start on `http://localhost:3000` (or the port specified in the `PORT` environment variable).

### Environment Variables

- `PORT` - Server port (default: 3000)
- `NEXT_PUBLIC_FOOBAR` - When set, displays a "foobar is set" banner at the top of the page

### Example with Environment Variable

```bash
NEXT_PUBLIC_FOOBAR=true npm start
```

This will display a banner at the top of the landing page indicating that the foobar environment variable is set.
