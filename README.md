# NovaSphere v2 Landing Experience

## English
### Overview
NovaSphere v2 is a fictional personal computing platform showcasing holographic experiences, quantum processing, and adaptive AI. This repository delivers the marketing landing site for Version 2.0, complete with hero storytelling, product positioning, and contact flows.

### Highlights
- Responsive hero with holographic device artwork and clear primary and secondary CTAs.
- Feature grid covering holographic interface, quantum processing, neural security, and infinite battery with on scroll animation.
- Pricing section for Standard, Pro, and Enterprise tiers, including a featured card callout.
- Product and Vision narrative with five year roadmap, stat callouts, and dual call to action buttons.
- Contact form with inline validation placeholders and client side confirmation.
- Sticky header navigation that stays readable on desktop and adapts to mobile.

### Pages
- `index.html`: Main landing page with hero, features, pricing, product vision, about, and contact sections.
- `team.html`: Dedicated team introduction with grid layout and hover interactions.
- `header.html`: Standalone demo of the enhanced sticky header, dropdown navigation, and breadcrumb trail.

### How to Run Locally
- Download or clone the repository into your workspace.
- Open `index.html` directly in any modern browser to preview the landing flow.
- Use a lightweight server such as `npx serve` or the VS Code Live Server extension if you prefer hot reload.
- Check `team.html` and `header.html` the same way to review auxiliary pages.

### Customize
- Update marketing copy and pricing data inside the relevant HTML sections.
- Replace placeholder images under `images/` with brand visuals or product renders.
- Adjust colors, typography, and layout tokens through `styles.css`.
- Extend animations or form handling logic inside `script.js` to match production needs.

### Assets and Credits
- Google Fonts `Montserrat` (weights 300, 400, 600, 700) powers the visual identity.
- SVG logo and product illustrations live in `images/`.
- Emoji icons decorate feature cards and product vision callouts.
- Contact details and address blocks reflect fictional NovaSphere information.
- JavaScript adds smooth anchor scrolling, feature card animation, and a simulated contact form submission.
