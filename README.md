## Getting Started
To view this landing page locally, simply follow these steps:
1.  **Clone the repository:**
    ```bash
    git clone <repository-url>
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd novashpere-landing-page
    ```
3.  **Open `index.html`:** Open the `index.html` file in your preferred web browser. No special server setup is required for basic viewing.
