# NovaSphere v2 - Landing Page

A modern, responsive landing page for NovaSphere, a futuristic personal computing device featuring holographic interfaces and quantum processing technology.

## Overview

NovaSphere v2 is a showcase landing page demonstrating a conceptual next-generation computing platform. This project features a clean, professional design with smooth animations, responsive layouts, and interactive elements.

## Features

- **Responsive Design**: Fully responsive layout that works seamlessly across desktop, tablet, and mobile devices
- **Modern UI/UX**: Clean and intuitive interface with smooth scrolling and animations
- **Interactive Elements**: Animated feature cards, hover effects, and form validation
- **Multi-Page Structure**: Includes main landing page and dedicated team page
- **Product Showcase**: Detailed sections for features, pricing, product vision, and company information
- **Contact Form**: Interactive contact form with client-side validation

## Technology Stack

- **HTML5**: Semantic markup with accessibility considerations
- **CSS3**: Modern styling with Flexbox/Grid layouts, animations, and transitions
- **JavaScript (Vanilla)**: Clean, dependency-free JavaScript for interactions
- **Google Fonts**: Montserrat font family for typography
- **SVG Graphics**: Scalable vector graphics for logos and product images

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- Optional: Local web server for development

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd /vercel/sandbox
```

2. Open in browser:
   - Simply open `index.html` in your web browser, or
   - Use a local development server:
   ```bash
   # Python 3
   python -m http.server 8000

   # Node.js (with http-server)
   npx http-server
   ```

3. Navigate to `http://localhost:8000` in your browser

## Usage

### Navigation

- Click navigation links to smoothly scroll to page sections

To customize the landing page:

2. **Modify Styles**: Edit `styles.css` to adjust colors, fonts, and layouts
3. **Add Features**: Extend `script.js` to add new interactive functionality
4. **Replace Images**: Update SVG files in the `images/` directory
