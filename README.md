# NovaSphere v2 - Landing Page

A modern, responsive landing page for NovaSphere v2, showcasing a revolutionary holographic computing device. This project demonstrates professional web design with smooth animations, intuitive navigation, and a compelling product presentation.

## Overview

NovaSphere v2 is a fictional next-generation personal computing device featuring:
- **Holographic Interface** - Interact with digital content in 3D
- **Quantum Processing** - Revolutionary performance with minimal power consumption
- **Neural Security** - Advanced biometric authentication
- **Infinite Battery** - Weeks of battery life through energy harvesting

## Features

- **Responsive Design** - Optimized for desktop, tablet, and mobile devices
- **Modern UI** - Clean, minimalist aesthetic with engaging interactions
- **Multiple Sections**:
  - Hero section with call-to-action buttons
  - Features showcase
  - Pricing plans (Standard, Pro, Enterprise)
  - Product vision and roadmap
  - About section
  - Contact form
- **Smooth Animations** - CSS transitions and JavaScript interactions
- **Navigation** - Header navigation with smooth scrolling links

## Getting Started

### Requirements
- Modern web browser (Chrome, Firefox, Safari, Edge)
- No build tools or dependencies required

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd novasphere-landing
```

2. Open `index.html` in your browser or serve with a local server:
```bash
# Python 3
python -m http.server 8000

# Node.js
npx http-server
```

3. Visit `http://localhost:8000` (or your server port)

## File Descriptions

- **index.html** - Main landing page with all product information and pricing
- **header.html** - Reusable header component with navigation
- **team.html** - Company/team information page
- **styles.css** - Global styling, layout, and animations
- **script.js** - Interactive features and form handling

## Customization

### Colors & Branding
Edit `styles.css` to modify:
- Color scheme (CSS variables at the top of the file)
- Typography and fonts
- Spacing and layout

### Content
Update HTML content in:
- `index.html` - Main page copy, features, pricing
- `header.html` - Navigation links
- `team.html` - Team information

### Images
Replace placeholder images in the `images/` directory:
- `logo.svg` - Company logo
- `product.svg` - Product images
