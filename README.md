# NovaSphere v2 — The Future of Personal Computing

NovaSphere v2 is a static marketing site that showcases an imaginary product focused on immersive, holographic personal computing.

This repository contains the landing pages and supporting assets used for the NovaSphere v2 site (HTML, CSS, images and a small script).

**Highlights**
- Holographic interface demo and product overview
- Quantum processing and neural security features described
- Pricing plans (Standard, Pro, Enterprise) showcased
- Product vision, team page, and contact form

Files of interest
- `index.html` — Main landing page with features, pricing, vision, contact
- `header.html` — Enhanced header/navigation demo and mobile menu
- `team.html` — Team page with bios for core members
- `styles.css`, `script.js`, `images/` — Styles, interactivity and assets

Quick preview
- Open `index.html` in your browser to preview the landing page locally.
- Or run a simple static server from the repo root:
  - `python -m http.server 8000`  (then open `http://localhost:8000`)
  - or `npx serve` if you prefer a Node-based static server

Development notes
- The site is static and intentionally small — no build step is required.
- `script.js` contains small UI helpers (mobile menu toggle, smooth scrolling).
- `styles.css` contains the visual system shared across pages.

Content summary
- Features: holographic UI, quantum processing, neural security, spatial audio, adaptive AI, long battery life.
- Pricing: Standard ($999), Pro ($1,499 — most popular), Enterprise ($2,499).
- Vision: an open platform for developers, artists and educators to build holographic experiences.
- Team: short bios for CEO & Founder, CTO, Head of Research, and Design Director.
