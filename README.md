# NovaSphere - The Future of Personal Computing

This repository contains the landing page for NovaSphere, an imaginary product that features holographic interfaces and advanced computing technologies.

## HTML Files Overview

### index.html
The main landing page for NovaSphere v2. This comprehensive single-page website includes:

- **Header Navigation**: Logo and navigation menu with links to Features, Pricing, About, and Contact sections
- **Hero Section**: Eye-catching introduction with headline "The Future of Personal Computing - Version 2.0", product description, and call-to-action buttons for demo and pre-order
- **Features Section**: Grid layout showcasing four key features:
  - Holographic Interface (3D digital content interaction)
  - Quantum Processing (high-performance quantum chip)
  - Neural Security (biometric authentication)
  - Infinite Battery (energy harvesting technology)
- **Pricing Section**: Three pricing tiers (Standard $999, Pro $1,499, Enterprise $2,499) with detailed feature lists
- **About Section**: Company background and philosophy
- **Contact Section**: Contact form with name, email, and message fields
- **Footer**: Comprehensive footer with logo, multiple link groups (Product, Company, Contact Info, Legal), and copyright information

Uses Montserrat font from Google Fonts and includes responsive design elements.

### header.html
A demonstration page showcasing an enhanced header navigation system with advanced features:

- **Enhanced Sticky Header**: Gradient purple background that stays visible while scrolling
- **Dropdown Navigation**: Two dropdown menus:
  - Products dropdown (Features, Pricing Plans, Tech Specs, Documentation)
  - Company dropdown (About Us, Our Team, Careers, News & Press)
- **Mobile Responsive**: Collapsible hamburger menu for mobile devices
- **Breadcrumb Navigation**: Shows current page location in the site hierarchy
- **Active States**: Highlights the current page in navigation
- **Smooth Animations**: Hover effects and transitions on navigation items
- **Demo Content**: Explains the navigation features and lists available subpages

Includes inline CSS for the enhanced header styling and JavaScript for mobile menu toggle functionality.

### team.html
A dedicated team page showcasing the NovaSphere leadership:

- **Team Grid Layout**: Responsive grid displaying team member cards
- **Team Members**: Features four key personnel:
  - Sarah Chen - CEO & Founder (former MIT quantum computing researcher)
  - Marcus Rodriguez - Chief Technology Officer (15 years holographic interface experience)
  - Dr. Aisha Patel - Head of Research (neural networks and machine learning expert)
  - James Wilson - Design Director (former Apple lead designer)
- **Card Design**: Each member has a circular profile image, name, position, and bio
- **Hover Effects**: Cards lift on hover for interactive feedback
- **Standard Navigation**: Uses the same header and footer as index.html with links back to main sections

Uses placeholder images from via.placeholder.com for team member photos.
