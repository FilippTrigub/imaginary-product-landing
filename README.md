# NovaSphere v2 Landing Experience

NovaSphere v2 presents a speculative personal computing device built around holographic interfaces, quantum processing, neural-grade security, and week-long power reserves. This repository hosts the marketing site that introduces the product story, pricing tiers, team, and enhanced navigation patterns for the NovaSphere brand.

## Experience Highlights
- Immersive hero showcase with call-to-action flows for demo viewing and pre-orders.
- Feature grid detailing holographic interface, quantum chip performance, adaptive security, and long-life energy harvesting.
- Pricing section with Standard, Pro, and Enterprise tiers, including badges, upsell messaging, and call-to-action buttons.
- Product & vision narrative outlining near-term capabilities, five-year platform goals, and supporting stats.
- Contact form wired with JavaScript to deliver instant acknowledgment before resetting the fields.
- Supplemental `team.html` profile page and `header.html` navigation demo that expand the story beyond the main landing page.

## Project Structure
- `index.html` — primary landing page with hero, feature, pricing, vision, about, and contact sections.
- `team.html` — dedicated team page showcasing leadership bios and roles.
- `header.html` — enhanced header and breadcrumb demo with dropdown navigation patterns.
- `styles.css` — global styling, responsive layout rules, and gradient-driven theming.
- `script.js` — smooth-scroll handling, contact form feedback, and viewport-triggered feature card animation.
- `images/` — brand assets (logo, product illustration) referenced by the layouts.

## Getting Started
1. Clone or download this repository.
2. Open `index.html` directly in a browser, or serve the directory with a lightweight static server (for example: `npx serve .`).
3. Visit `team.html` and `header.html` to explore the extended navigation and team experience.

## Design & Implementation Notes
- Typography relies on Google Fonts (`Montserrat` family); an internet connection is required to load the font files.
- The layout is responsive down to tablet and mobile breakpoints, including sticky navigation and mobile-friendly buttons.
- JavaScript enhancements are framework-free and scoped to smooth scrolling, in-page animations, and basic client-side form handling.
- Pricing cards and feature modules leverage subtle gradients, shadows, and hover effects defined in `styles.css`.

Feel free to adapt the copy, visuals, or interactive behavior to plug NovaSphere v2 into your own storytelling or prototype needs.
