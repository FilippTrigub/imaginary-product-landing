# Hello World Scripts

This repository contains "Hello, world!" programs in multiple languages.

Run examples:

- Bash: `./hello.sh`
- Node.js: `./hello.js` (requires Node.js)
- Python: `./hello.py` (requires Python 3)
- Ruby: `./hello.rb` (requires Ruby)
- Go: `go run hello.go` (requires Go toolchain)
- Rust: `rustc hello.rs && ./hello` (requires Rust toolchain)
- Java: `javac Hello.java && java Hello` (requires JDK)
- C: `cc hello.c -o hello_c && ./hello_c`
- C++: `c++ hello.cpp -o hello_cpp && ./hello_cpp`
- PHP: `./hello.php` (requires PHP CLI)
- Perl: `./hello.pl` (requires Perl)
- Haskell: `runhaskell hello.hs` (requires GHC)
