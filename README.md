## 🛠️ Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, or Edge)
- Internet connection for optimal holographic rendering

### Installation

1. Clone this repository:
   ```bash
   git clone https://github.com/yourusername/novosphere-landing.git
   cd novosphere-landing
   ```

2. Open `index.html` in your web browser to view the landing page.

### Local Development

For development purposes, you can serve the files locally using any static server. For example, with Python:

```bash
python -m http.server 8000
```

Then navigate to `http://localhost:8000` in your browser.
