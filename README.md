# NovaSphere v2.0

A modern, responsive landing page showcasing NovaSphere - a futuristic personal computing device with holographic interface technology.

## Overview

NovaSphere is a concept product landing page demonstrating advanced web design techniques with a focus on clean aesthetics, smooth animations, and user engagement. The site presents an imaginary revolutionary computing device that blends holographic displays with quantum processing technology.

## Features

### Product Highlights
- **Holographic Interface** - 3D interactive digital content
- **Quantum Processing** - Proprietary quantum chip technology
- **Neural Security** - Adaptive biometric authentication
- **Infinite Battery** - Energy harvesting with weeks of runtime

### Website Features
- Fully responsive design (mobile, tablet, desktop)
- Smooth scrolling navigation
- Interactive pricing plans (Standard, Pro, Enterprise)
- Contact form

## Getting Started

### Prerequisites

No build tools or dependencies required. This is a static HTML/CSS/JavaScript website.

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd novasphere
```

2. Open the website:

Simply open `index.html` in your web browser, or use a local development server:

```bash
# Using Python 3
python3 -m http.server 8000

# Using Node.js (http-server)
npx http-server -p 8000

# Using PHP
php -S localhost:8000
```

3. Navigate to `http://localhost:8000` in your browser
