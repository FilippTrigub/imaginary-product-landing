# NovaSphere v2 Project

## Overview
NovaSphere v2 is a project focused on enhancing personal computing through innovative technology. The project includes a modern website that showcases the features, pricing, and team behind NovaSphere.

## HTML Files

### 1. `header.html`
This file contains the enhanced header navigation for the NovaSphere website. Key features include:
- **Sticky Header**: The header remains visible while scrolling.
- **Dropdown Menus**: Navigation links for Products and Company sections with subpage links.
- **Mobile Responsiveness**: A collapsible mobile menu for smaller screens.
- **Breadcrumb Navigation**: Indicates the current page location.
- **Smooth Animations**: Hover effects and transitions for a better user experience.

### 2. `index.html`
The main landing page for NovaSphere v2. It features:
- **Hero Section**: Introduces the product with a call-to-action to watch a demo or pre-order.
- **Key Features**: Highlights the main features of the NovaSphere device, including holographic interface, quantum processing, and neural security.
- **Pricing Plans**: Details on different pricing tiers for the NovaSphere device.
- **About Section**: Information about the company and its mission.
- **Contact Form**: Allows users to get in touch with the company.

### 3. `team.html`
This file presents the team behind NovaSphere. It includes:
- **Team Section**: Displays team members with their roles and bios.
- **Grid Layout**: A responsive grid layout for showcasing team members.
- **Hover Effects**: Subtle animations when hovering over team member cards.

## Conclusion
The NovaSphere v2 project aims to revolutionize personal computing with its innovative features and user-friendly design. The HTML files provide a structured and visually appealing way to present the product and its team.
