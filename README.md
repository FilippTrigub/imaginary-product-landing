# NovaSphere v2 - The Future of Personal Computing

A modern, responsive landing page for NovaSphere, an imaginary revolutionary computing device featuring holographic interfaces, quantum processing, and advanced AI capabilities.

## 🌟 Overview

NovaSphere v2 is a conceptual product website showcasing cutting-edge personal computing technology. This project demonstrates modern web design principles with a clean, professional interface built using HTML, CSS, and JavaScript.

## ✨ Features

- **Responsive Design**: Fully responsive layout that works seamlessly across desktop, tablet, and mobile devices
- **Modern UI/UX**: Clean, professional design with smooth animations and transitions
- **Interactive Navigation**: Enhanced header with dropdown menus and breadcrumb navigation
- **Multiple Pages**:
  - `index.html` - Main landing page with product features, pricing, and contact form
  - `team.html` - Team member profiles page
  - `header.html` - Enhanced navigation demo page
- **Product Sections**:
  - Hero section with call-to-action buttons
  - Feature showcase with 4 key product features
  - Pricing plans (Standard, Pro, Enterprise)
  - Product vision and 5-year roadmap
  - About section
  - Contact form
- **Visual Assets**: Custom SVG logo and product images

## 🚀 Technologies Used

- **HTML5**: Semantic markup structure
- **CSS3**: Modern styling with Flexbox and Grid layouts
- **JavaScript**: Interactive navigation and smooth scrolling
- **Google Fonts**: Montserrat font family for typography
- **SVG Graphics**: Scalable vector images for logo and product visuals
