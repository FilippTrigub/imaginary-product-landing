# NovaSphere v2 - Landing Page

A modern, responsive landing page for NovaSphere, a futuristic personal computing device featuring holographic interfaces and advanced quantum processing technology.

## Overview

This project showcases a product landing page for an imaginary cutting-edge technology product. The site demonstrates modern web design principles with clean layouts, smooth interactions, and responsive design that works across all devices.

## Features

- **Responsive Design**: Fully responsive layout that adapts to mobile, tablet, and desktop screens
- **Modern UI**: Clean, professional design with smooth animations and transitions
- **Product Showcase**: Comprehensive feature descriptions and pricing information
- **Contact Form**: Interactive contact form for user inquiries
- **ASCII Art**: Creative ASCII art branding in HTML comments

## Technology Stack

- **HTML5**: Semantic markup with accessibility features
- **CSS3**: Modern styling with custom properties, flexbox, and grid layouts
- **JavaScript**: Interactive components and form handling
- **Google Fonts**: Montserrat font family for typography

## Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- Optional: A local web server for development

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd novasphere
```

2. Open the project:
   - Simply open `index.html` in your browser, or
   - Use a local server for better development experience:

```bash
# Using Python 3
python3 -m http.server 8000

# Using Node.js (with npx)
npx serve

# Using PHP
php -S localhost:8000
```

3. Navigate to `http://localhost:8000` in your browser

## Development

### File Structure

- **index.html**: Main landing page with all sections
- **header.html**: Standalone header component with ASCII art
- **team.html**: Team/about page
- **styles.css**: All styling including responsive breakpoints
- **script.js**: Form handling and interactive features

### Customization

To customize the landing page:

1. **Colors**: Update CSS custom properties in `styles.css`
2. **Content**: Edit HTML files to change text and structure
3. **Images**: Replace images in the `/images` directory
4. **Fonts**: Modify the Google Fonts import in HTML files
