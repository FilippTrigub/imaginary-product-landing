# NovaSphere v2.0 - Landing Page

A modern, responsive landing page for NovaSphere, a fictional revolutionary personal computing device featuring holographic interfaces and quantum processing technology.

## 🚀 Overview

NovaSphere v2.0 is a fully responsive HTML/CSS/JavaScript website showcasing an imaginary next-generation computing product. The site features a clean, modern design with smooth animations and an intuitive user experience.

## ✨ Features

- **Responsive Design**: Fully optimized for desktop, tablet, and mobile devices
- **Modern UI/UX**: Clean interface with smooth transitions and hover effects
- **Multiple Sections**:
  - Hero section with product showcase
  - Key features grid with icons
  - Pricing plans (Standard, Pro, Enterprise)
  - Product vision and statistics
  - About section
  - Contact form
  - Team page
- **Custom Typography**: Uses Montserrat font family from Google Fonts
- **SVG Graphics**: Scalable logo and product images
- **Interactive Elements**: Smooth scrolling navigation and form interactions

## 🚀 Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- A local web server (optional, for development)

### Installation

1. Clone or download this repository:
   ```bash
   git clone <repository-url>
   cd /vercel/sandbox
   ```

2. Open the project in your preferred code editor

3. Serve the files using a local web server:
   
   **Option 1: Using Python**
   ```bash
   # Python 3
   python3 -m http.server 8000
   ```
   
   **Option 2: Using Node.js (http-server)**
   ```bash
   npx http-server -p 8000
   ```
   
   **Option 3: Using PHP**
   ```bash
   php -S localhost:8000
   ```

4. Open your browser and navigate to:
   ```
   http://localhost:8000
   ```

### Direct File Access

Alternatively, you can simply open `index.html` directly in your browser, though some features may work better with a local server.

## 🎨 Customization

### Colors
The color scheme can be modified in `styles.css`. Key color variables are used throughout:
- Primary: `#6366f1` (Indigo)
- Secondary: Various shades of gray
- Accent colors for hover states

### Typography
The site uses Montserrat font. To change fonts, update the Google Fonts import in the HTML files and modify the `font-family` declarations in `styles.css`.

### Content
- Update text content directly in `index.html` and `team.html`
- Replace images in the `/images` directory
- Modify pricing plans, features, and team members as needed

## 📱 Responsive Breakpoints

The site is optimized for the following breakpoints:
- Desktop: 1200px and above
- Tablet: 768px - 1199px
- Mobile: Below 768px
