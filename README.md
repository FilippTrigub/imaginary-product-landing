## Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- Basic web server for local development (optional)

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd novasphere
```

2. Open in browser:
   - **Option 1**: Double-click `index.html` to open directly
   - **Option 2**: Use a local web server:
   ```bash
   # Using Python
   python -m http.server 8000

   # Using Node.js
   npx http-server
   ```

3. Navigate to `http://localhost:8000` (if using a web server)

## Customization

### Colors
Edit CSS custom properties in `styles.css`:
```css
:root {
    --primary-color: #6c5ce7;
    --secondary-color: #00b894;
    --dark-color: #2d3436;
    --light-color: #dfe6e9;
}
```

### Content
- Update text in `index.html`
- Replace images in `images/` directory
- Modify pricing in the pricing section
- Add new sections as needed

### Styling
- Edit `styles.css` for visual changes
- Modify responsive breakpoints
- Adjust animations and transitions
