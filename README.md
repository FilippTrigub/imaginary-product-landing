# NovaSphere - The Future of Personal Computing

This repository contains the landing page for NovaSphere, an imaginary product that features holographic interfaces and advanced computing technologies.

## Beginner Coding Exercises: JavaScript Basics

Below is a mini workbook of beginner-friendly JavaScript exercises you can complete directly in a browser console or any online editor. Each exercise introduces one concept, builds gradually, and includes a suggestion for optional extensions.

### 1. Hello, Solar System
- **Goal:** Write a function `greetPlanet(name)` that logs `"Hello, <name>!"`.
- **Concepts:** Functions, string interpolation.
- **Stretch:** If `name` is empty, default to `"Mars"`.

### 2. Nova Counter
- **Goal:** Create a loop that logs the numbers 1 through 20.
- **Concepts:** `for` loops, counters.
- **Stretch:** Replace multiples of 5 with `"Nova"`.

### 3. Temperature Translator
- **Goal:** Convert Celsius to Fahrenheit using a function.
- **Concepts:** Math operations, return values.
- **Stretch:** Add the reverse conversion.

### 4. Battery Level Checker
- **Goal:** Given a percentage, log whether the battery is `"Full"`, `"Medium"`, or `"Low"`.
- **Concepts:** `if/else` statements.
- **Stretch:** Accept thresholds as parameters.

### 5. Hologram Playlist
- **Goal:** Store three favorite songs in an array and print them.
- **Concepts:** Arrays, indexing.
- **Stretch:** Add a function to add a song and reprint.

### 6. Crew Roster Search
- **Goal:** Given an array of crew names, write a function that returns `true` if a name exists.
- **Concepts:** Array methods, boolean returns.
- **Stretch:** Make search case-insensitive.

### 7. Mission Cost Calculator
- **Goal:** Sum the numbers in an array to calculate total mission cost.
- **Concepts:** Loops, accumulators.
- **Stretch:** Skip negative values to avoid invalid entries.

### 8. Beacon Password Validator
- **Goal:** Check if a password is at least 8 characters and contains a number.
- **Concepts:** String length, regex or iteration.
- **Stretch:** Add special-character requirements.

### 9. AI Status Dashboard
- **Goal:** Use an object to store status fields like `mode`, `uptime`, `temperature` and log them.
- **Concepts:** Objects, property access.
- **Stretch:** Create a function that updates a field and re-logs status.

### 10. Launch Countdown Timer
- **Goal:** Build a countdown from 10 to 0 using `setInterval`, logging each number.
- **Concepts:** Timers, callback functions.
- **Stretch:** When reaching 0, display `"Launch!"` and clear the interval.

### 11. Signal Strength Analyzer
- **Goal:** Given an array of signal readings, filter out values below 50.
- **Concepts:** Array filtering.
- **Stretch:** Compute the average of the remaining signals.

### 12. Nova Dice Simulator
- **Goal:** Simulate rolling a six-sided die five times and store results.
- **Concepts:** Random numbers, loops.
- **Stretch:** Count how many times each face appears.

### 13. Space Budget Tracker
- **Goal:** Track daily spend in an array and determine the highest expense.
- **Concepts:** Array iteration, comparison.
- **Stretch:** Return both value and day index.

### 14. Message Encoder
- **Goal:** Replace every vowel in a string with `*`.
- **Concepts:** String manipulation.
- **Stretch:** Allow the replacement character to be a parameter.

### 15. Docking Queue Manager
- **Goal:** Use an array to model ships waiting to dock; add arrivals and remove departures using `push`/`shift`.
- **Concepts:** Queue operations.
- **Stretch:** Display the queue length after each change.
