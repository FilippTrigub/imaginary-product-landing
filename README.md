# NovaSphere v2 - The Future of Personal Computing

This repository contains the source code for the landing page of NovaSphere v2, a revolutionary personal computing device with a holographic interface.

## 🚀 Features

*   **Holographic Interface:** Interact with digital content in three dimensions.
*   **Quantum Processing:** Unprecedented performance with a proprietary quantum chip.
*   **Neural Security:** Advanced biometric authentication that adapts to you.
*   **Infinite Battery:** Revolutionary energy harvesting technology.

## 📄 Pages

The website consists of the following pages:

*   **`index.html`:** The main landing page, showcasing the product's features, pricing, and vision.
*   **`team.html`:** Introduces the brilliant minds behind NovaSphere.
*   **`header.html`:** A reusable header component with enhanced navigation, including dropdown menus and a mobile-responsive design.

## ሩ Running the Project

To view the website, simply open the `index.html` file in your web browser.
