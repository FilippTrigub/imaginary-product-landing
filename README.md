# NovaSphere Landing Page

A modern, responsive landing page for NovaSphere, showcasing an innovative personal computing product with holographic interfaces and advanced technologies.

## Features

- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Interactive Elements**: Smooth animations and user interactions powered by JavaScript
- **Clean UI**: Modern design with custom CSS styling
- **Modular Structure**: Organized HTML components for easy maintenance

## Project Structure

```
/
├── index.html          # Main landing page
├── header.html         # Header component
├── team.html           # Team section component
├── styles.css          # Main stylesheet
├── script.js           # JavaScript for interactivity
├── images/
│   ├── logo.svg        # NovaSphere logo
│   └── product.svg     # Product illustration
└── README.md           # This file
```

## Getting Started

1. Clone the repository
2. Open `index.html` in your web browser
3. For development, you can use a local server (e.g., `python -m http.server` or Live Server extension)
