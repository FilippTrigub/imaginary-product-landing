# NovaSphere v2.0 - The Future of Personal Computing

A modern, responsive landing page showcasing NovaSphere, a revolutionary holographic computing device with quantum processing capabilities.

## 📋 Project Overview

NovaSphere v2.0 is a multi-page website featuring a futuristic product landing page with advanced features including holographic interfaces, quantum processing, neural security, and infinite battery technology. The site demonstrates modern web design principles with smooth animations, responsive layouts, and interactive elements.

## 🌟 Features

### Pages
- **index.html** - Main landing page with:
  - Hero section with animated product showcase
  - Key features grid (Holographic Interface, Quantum Processing, Neural Security, Infinite Battery)
  - Pricing plans (Standard, Pro, Enterprise)
  - Product & Vision section with statistics and roadmap
  - About section
  - Contact form
  
- **team.html** - Team page featuring:
  - Team member profiles with photos
  - Role descriptions and bios
  - Responsive grid layout
  
- **header.html** - Enhanced navigation demo with:
  - Dropdown menus for Products and Company sections
  - Breadcrumb navigation
  - Mobile-responsive collapsible menu
  - Sticky header with smooth scrolling
  - Active state highlighting

### Design Elements
- **Modern Gradient Design** - Purple/blue gradient theme throughout
- **Smooth Animations** - Floating product images, hover effects, scroll animations
- **Responsive Layout** - Mobile-first design that adapts to all screen sizes
- **Interactive Components** - Dropdown menus, smooth scrolling, form validation
- **Professional Typography** - Montserrat font family from Google Fonts

## 🛠️ Technologies Used

- **HTML5** - Semantic markup structure
- **CSS3** - Modern styling with:
  - CSS Grid and Flexbox layouts
  - CSS animations and transitions
  - Custom properties and gradients
  - Media queries for responsiveness
- **JavaScript (ES6+)** - Interactive features:
  - Smooth scrolling navigation

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No build tools or dependencies required - pure HTML/CSS/JS

### Installation

1. Clone or download the repository:
```bash
git clone <repository-url>
cd /vercel/sandbox
```

2. Open the project:
   - Simply open `index.html` in your web browser
   - Or use a local development server:
   
```bash
# Using Python 3
python3 -m http.server 8000

# Using Node.js (http-server)
npx http-server -p 8000

# Using PHP
php -S localhost:8000
```

3. Navigate to `http://localhost:8000` in your browser

## 📝 Customization

### Changing Colors
Edit the CSS variables or gradient values in `styles.css`:
```css
.btn-primary {
    background: linear-gradient(135deg, #6e8efb, #a777e3);
}
```

### Modifying Content
- Update text directly in HTML files
- Replace images in the `/images` folder
- Adjust pricing plans in the pricing section

### Adding New Pages
1. Create a new HTML file
2. Copy the header and footer from existing pages
3. Link to `styles.css` and `script.js`
4. Add navigation links in the header
