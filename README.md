# NovaSphere v2.0 - Landing Page

A modern, responsive landing page for NovaSphere, a fictional revolutionary personal computing device featuring holographic interfaces and quantum processing technology.

## 🚀 Overview

This project showcases a complete product landing page built with vanilla HTML, CSS, and JavaScript. It demonstrates modern web design principles, smooth animations, and responsive layouts suitable for marketing a cutting-edge technology product.

## ✨ Features

- **Responsive Design**: Fully responsive layout that works seamlessly across desktop, tablet, and mobile devices
- **Modern UI/UX**: Clean, professional design with smooth animations and transitions
- **Interactive Elements**: Smooth scrolling navigation, animated feature cards, and form validation
- **Multi-page Structure**: Includes main landing page and dedicated team page
- **Product Showcase**: Comprehensive sections including:
  - Hero section with call-to-action buttons
  - Feature highlights with icon-based cards
  - Pricing tiers (Standard, Pro, Enterprise)
  - Product vision and roadmap
  - About section
  - Contact form
  - Team member profiles

## 🛠️ Technologies Used

- **HTML5**: Semantic markup and accessibility features
- **CSS3**: Modern styling with Flexbox, Grid, animations, and transitions
- **JavaScript (ES6+)**: DOM manipulation, event handling, and smooth scrolling
- **Google Fonts**: Montserrat font family for typography

## 🎨 Design Highlights

- **Typography**: Montserrat font family with multiple weights (300, 400, 600, 700)
- **Color Scheme**: Professional gradient-based design with primary blues and purples
- **Animations**: Scroll-triggered fade-in animations for feature cards
- **Layout**: CSS Grid and Flexbox for responsive, modern layouts
- **Icons**: Emoji-based icons for simplicity and universal compatibility

## 🚦 Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- A local web server (optional, for best experience)

### Installation & Running

1. **Clone or download** this repository to your local machine

2. **Open directly in browser**:
   ```bash
   # Navigate to the project directory
   cd /vercel/sandbox
   
   # Open index.html in your default browser
   open index.html  # macOS
   xdg-open index.html  # Linux
   start index.html  # Windows
   ```

3. **Or use a local server** (recommended):
   ```bash
   # Using Python 3
   python3 -m http.server 8000
   
   # Using Node.js (if you have http-server installed)
   npx http-server -p 8000
   
   # Using PHP
   php -S localhost:8000
   ```
   
   Then open `http://localhost:8000` in your browser
