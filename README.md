# NovaSphere - The Future of Personal Computing

This repository contains the landing page for NovaSphere, an imaginary product that features holographic interfaces and advanced computing technologies.


## New pages added
- about.html — company story, values and mission
- contact.html — contact form with front-end validation (no server)
- blog.html — static blog listing

## New static assets and files
- footer.html — reusable footer partial
- images/favicon.ico — placeholder for favicon (replace with binary .ico file)
- robots.txt — allow all agents and point to sitemap
- sitemap.xml — basic sitemap with main pages

## Testing instructions
1. Open the HTML files directly in your browser (file://) or host them with a static file server of your choice.
2. Verify navigation links in the header work and lead to the correct pages.
3. On smaller screens, test the mobile menu button to ensure navigation toggles.
4. On contact.html, try submitting the contact form with missing fields to see validation messages. No server calls are made.

Note: I did not start any server or long-running process per your instruction.
