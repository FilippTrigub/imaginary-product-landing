# NovaSphere v2 Landing Page

NovaSphere v2 is a concept marketing site that imagines a holographic personal computing device. The project focuses on polished storytelling, responsive layout, and light client-side interactivity for a fictional launch campaign.

## Highlights
- Immersive hero section with calls to action for demo viewing and pre-orders
- Feature, pricing, and product-vision sections that showcase the NovaSphere ecosystem
- Smooth in-page navigation and scroll-triggered animations implemented in vanilla JavaScript
- Contact form prototype with client-side validation placeholder logic

## Tech Stack
- HTML5 for semantic page structure
- CSS3 with custom gradients, cards, and responsive grid layouts
- Vanilla JavaScript for smooth scrolling, form handling, and feature card animations
- Google Fonts (Montserrat) for typography

## Getting Started
1. Clone or download this repository.
2. Open `index.html` directly in a modern browser, or run a lightweight static server such as `npx serve .` to preview locally.
3. Explore the page and interact with navigation links, pricing cards, and the contact form.

## Project Structure
- `index.html` — Main landing page for NovaSphere v2
- `team.html` — Optional team spotlight page
- `header.html` — Reusable header markup (not yet wired into the main page)
- `styles.css` — Global styles, layout utilities, and responsive rules
- `script.js` — Smooth scrolling, feature animations, and contact form handler
- `images/` — SVG assets for logo and product visuals

## Customization Tips
- Update copy in `index.html` to tailor the narrative for your own product story.
- Replace assets inside `images/` to reflect your brand identity.
- Extend `script.js` with real form submission logic or additional animations as needed.
- Break styles into modules if the page grows beyond a single file.
