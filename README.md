# Qwen Code - Hello World Collection

Welcome to the Hello World Collection! This project contains various implementations of the classic "Hello, World!" program in different programming languages and technologies.

## Files Included

### Interactive Web Page
- `hello_world.html` - An interactive HTML page demonstrating Hello World in multiple ways:
  - HTML-based greeting
  - JavaScript alert and DOM manipulation
  - CSS pseudo-element interaction
  - Examples of Hello World in other languages displayed as code

### Standalone Scripts
- `hello.py` - Python implementation
- `hello.js` - Node.js/JavaScript implementation
- `hello.sh` - Bash shell script implementation

## How to Run

### Web Interface
Simply open `hello_world.html` in any web browser to see the interactive Hello World collection.

### Standalone Scripts

#### Python
```bash
python3 hello.py
```

#### Node.js/JavaScript
```bash
node hello.js
```

Or make it executable:
```bash
chmod +x hello.js
./hello.js
```

#### Bash
```bash
bash hello.sh
```

Or make it executable:
```bash
chmod +x hello.sh
./hello.sh
```

## Technologies Demonstrated

This collection showcases Hello World programs in:

- **HTML** - Basic markup
- **CSS** - Using pseudo-elements and hover effects
- **JavaScript** - DOM manipulation and alerts
- **Python** - Standard output program
- **Node.js/JavaScript** - Server-side JavaScript
- **Bash** - Shell scripting

## About

This collection serves as a fun introduction to various programming approaches and technologies. Each implementation is a simple, clean way of saying "Hello, World!" in the respective technology.
