## Getting Started

### Prerequisites

No external dependencies required. This is a vanilla HTML/CSS/JavaScript project.

### Running Locally

1. Clone the repository:
```bash
git clone <repository-url>
cd novasphere
```

2. Open `index.html` in your web browser or serve using a local HTTP server:
```bash
# Using Python 3
python -m http.server 8000

# Using Node.js (http-server)
npx http-server
```

3. Visit `http://localhost:8000` in your browser
