# NovaSphere

NovaSphere is a static landing page demo for an imaginary personal computing product. This repository contains the HTML, CSS, and JavaScript source for the site.

Overview
- Purpose: A small, static demo landing page showcasing a fictional product and UI concepts.
- Contents: `index.html`, `styles.css`, `script.js`, plus assets in the `images/` folder.

Quick Start
- Preview locally with a static server (Node): `npx serve .` and open `http://localhost:3000`.
- Or with Python 3: `python -m http.server 8000` and open `http://localhost:8000`.

Development Environment
- This project is plain HTML/CSS/JS and does not require a build step.
- Sandbox notes (if you're running in a controlled environment): the sandbox uses Amazon Linux 2023 with `dnf` and Node.js 22 available. Python 3 is present; if you need to install Python packages, ensure `pip` is installed first.

Project Structure
- `index.html` — main landing page
- `header.html` — reusable header fragment
- `styles.css` — site styles
- `script.js` — client-side behavior
- `images/` — image assets

Deploying
- Any static host will work (Vercel, Netlify, GitHub Pages, etc.).
- On Vercel: import the repository and use the default static settings.
