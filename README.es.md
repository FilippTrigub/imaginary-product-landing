# NovaSphere v2.0 - Página de Aterrizaje

Una página de aterrizaje moderna y responsiva para NovaSphere, un dispositivo revolucionario de computación personal ficticio que presenta interfaces holográficas y tecnología de procesamiento cuántico.

## 🚀 Descripción General

Este proyecto muestra una página de aterrizaje de producto completa construida con HTML, CSS y JavaScript vanilla. Demuestra principios modernos de diseño web, animaciones suaves y diseños responsivos adecuados para comercializar un producto tecnológico de vanguardia.

## ✨ Características

### Páginas
- **Página de Aterrizaje Principal** (`index.html`) - Escaparate completo del producto con:
  - Sección hero con botones de llamada a la acción
  - Destacados de características con tarjetas animadas
  - Niveles de precios (Estándar, Pro, Empresarial)
  - Visión del producto y estadísticas
  - Sección Acerca de
  - Formulario de contacto
  
- **Página del Equipo** (`team.html`) - Conoce al equipo detrás de NovaSphere con:
  - Perfiles de miembros del equipo
  - Descripciones de roles
  - Animaciones al pasar el cursor

### Funcionalidad
- **Desplazamiento Suave** - Navegación fluida entre secciones de la página
- **Manejo de Formularios** - Formulario de contacto con validación y manejo de envío
- **Animaciones de Desplazamiento** - Las tarjetas de características se animan al aparecer en pantalla
- **Diseño Responsivo** - Diseño completamente responsivo para todos los tamaños de dispositivos
- **UI/UX Moderna** - Diseño limpio y profesional con la familia de fuentes Montserrat

## 🛠️ Stack Tecnológico

- **HTML5** - Estructura de marcado semántico
- **CSS3** - Estilo moderno con diseños flexbox/grid
- **JavaScript (ES6+)** - Características interactivas y animaciones
- **Google Fonts** - Familia de fuentes Montserrat
- **Gráficos SVG** - Logo e imágenes de producto escalables

## 🚦 Comenzando

### Requisitos Previos
- Un navegador web moderno (Chrome, Firefox, Safari, Edge)
- Opcional: Servidor web local para desarrollo

### Instalación y Ejecución

1. **Clonar o descargar** este repositorio

2. **Abrir directamente en el navegador:**
   ```bash
   # Simplemente abre index.html en tu navegador
   open index.html
   ```

3. **O usar un servidor local** (recomendado):
   ```bash
   # Usando Python 3
   python3 -m http.server 8000
   
   # Usando Node.js (http-server)
   npx http-server -p 8000
   
   # Usando PHP
   php -S localhost:8000
   ```

4. **Navegar a:**
   ```
   http://localhost:8000
   ```

## 🎨 Personalización

### Actualizar Contenido
- **Información del Producto**: Edita el contenido de texto en `index.html`
- **Planes de Precios**: Modifica las tarjetas de precios en la sección `#pricing`
- **Miembros del Equipo**: Actualiza los perfiles en `team.html`
- **Información de Contacto**: Cambia los detalles del pie de página en ambos archivos HTML

### Estilo
- **Colores**: Actualiza las variables CSS o valores de color en `styles.css`
- **Fuentes**: Cambia la importación de Google Fonts en las secciones `<head>` del HTML
- **Diseño**: Modifica las propiedades de grid/flexbox en `styles.css`

### Imágenes
- Reemplaza `images/logo.svg` con tu propio logo
- Reemplaza `images/product.svg` con la imagen de tu producto
- Actualiza las imágenes de marcador de posición de los miembros del equipo en `team.html`
