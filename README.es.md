# NovaSphere v2.0

> El Futuro de la Computación Personal

Una página de destino moderna y responsiva que presenta NovaSphere — un dispositivo informático revolucionario imaginario con interfaces holográficas, procesamiento cuántico y capacidades avanzadas de IA.

## 🌟 Descripción General

NovaSphere v2.0 es una página de destino conceptual de producto que demuestra principios de diseño web de vanguardia y técnicas modernas de HTML/CSS/JavaScript. El sitio presenta un dispositivo informático holográfico ficticio con una experiencia de usuario inmersiva y profesional.

## 🚀 Comenzando

### Requisitos Previos
- Un navegador web moderno (Chrome, Firefox, Safari, Edge)
- Opcional: Servidor web local para desarrollo

### Instalación

1. **Clonar el repositorio**
   ```bash
   git clone <url-del-repositorio>
   cd /vercel/sandbox
   ```

2. **Abrir en el navegador**
   
   Simplemente abre `index.html` en tu navegador web:
   ```bash
   # Usando el navegador predeterminado
   open index.html
   
   # O con un navegador específico
   firefox index.html
   chrome index.html
   ```

3. **Opcional: Ejecutar con servidor local**
   
   Para una mejor experiencia de desarrollo:
   ```bash
   # Python 3
   python3 -m http.server 8000
   
   # Node.js (con http-server)
   npx http-server -p 8000
   ```
   
   Luego visita `http://localhost:8000`
