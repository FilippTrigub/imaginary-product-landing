# Page d'Accueil NovaSphere

Une page d'accueil moderne et responsive pour NovaSphere, présentant un produit informatique personnel innovant avec des interfaces holographiques et des technologies avancées.

## Fonctionnalités

- **Design Responsive** : Optimisé pour les ordinateurs de bureau, tablettes et appareils mobiles
- **Éléments Interactifs** : Animations fluides et interactions utilisateur alimentées par JavaScript
- **Interface Épurée** : Design moderne avec un style CSS personnalisé
- **Structure Modulaire** : Composants HTML organisés pour une maintenance facile

## Structure du Projet

```
/
├── index.html          # Page d'accueil principale
├── header.html         # Composant d'en-tête
├── team.html           # Composant de la section équipe
├── styles.css          # Feuille de style principale
├── script.js           # JavaScript pour l'interactivité
├── images/
│   ├── logo.svg        # Logo NovaSphere
│   └── product.svg     # Illustration du produit
└── README.md           # Ce fichier
```

## Démarrage

1. Clonez le dépôt
2. Ouvrez `index.html` dans votre navigateur web
3. Pour le développement, vous pouvez utiliser un serveur local (par exemple, `python -m http.server` ou l'extension Live Server)
