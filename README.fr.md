# NovaSphere v2 Page d'atterrissage

NovaSphere v2 est un site marketing conceptuel qui imagine un appareil informatique personnel holographique. Le projet se concentre sur une narration soignée, une mise en page réactive et une légère interactivité côté client pour une campagne de lancement fictive.

## Points forts
- Section héros immersive avec des appels à l'action pour la visualisation de la démo et les précommandes
- Sections sur les fonctionnalités, les prix et la vision du produit qui présentent l'écosystème NovaSphere
- Navigation fluide dans la page et animations déclenchées par le défilement implémentées en JavaScript vanilla
- Prototype de formulaire de contact avec une logique d'espace réservé de validation côté client

## Pile technologique
- HTML5 pour la structure de page sémantique
- CSS3 avec des dégradés personnalisés, des cartes et des mises en page de grille réactives
- JavaScript vanilla pour le défilement fluide, la gestion des formulaires et les animations des cartes de fonctionnalités
- Google Fonts (Montserrat) pour la typographie

## Pour commencer
1. Clonez ou téléchargez ce référentiel.
2. Ouvrez `index.html` directement dans un navigateur moderne, ou exécutez un serveur statique léger tel que `npx serve .` pour prévisualiser localement.
3. Explorez la page et interagissez avec les liens de navigation, les cartes de prix et le formulaire de contact.

## Structure du projet
- `index.html` — Page d'atterrissage principale pour NovaSphere v2
- `team.html` — Page facultative de présentation de l'équipe
- `header.html` — Balisage d'en-tête réutilisable (pas encore câblé dans la page principale)
- `styles.css` — Styles globaux, utilitaires de mise en page et règles réactives
- `script.js` — Défilement fluide, animations de fonctionnalités et gestionnaire de formulaire de contact
- `images/` — Actifs SVG pour le logo et les visuels du produit

## Conseils de personnalisation
- Mettez à jour la copie dans `index.html` pour adapter le récit à votre propre histoire de produit.
- Remplacez les actifs à l'intérieur de `images/` pour refléter l'identité de votre marque.
- Étendez `script.js` avec une logique de soumission de formulaire réelle ou des animations supplémentaires selon vos besoins.
- Divisez les styles en modules si la page dépasse un seul fichier.

## Contribuer
Ce référentiel sert d'invite créative et de ressource d'apprentissage. Fourchez-le, adaptez les composants et itérez sur la conception pour expérimenter des concepts de page d'atterrissage avancés.
