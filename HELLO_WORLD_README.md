# Hello World Scripts Collection

This directory contains multiple "Hello World" implementations in different languages and formats.

## Available Scripts

### 1. JavaScript (Node.js) - `hello-world.js`
Basic JavaScript Hello World with multiple implementation patterns.

**Run:**
```bash
node hello-world.js
```

**Features:**
- Simple console.log
- Function-based approach
- Arrow function
- Class-based implementation

---

### 2. HTML Page - `hello-world.html`
Interactive HTML page with styled Hello World message.

**Run:**
Open in a web browser:
```bash
# If you have a simple HTTP server
python3 -m http.server 8000
# Then visit: http://localhost:8000/hello-world.html
```

**Features:**
- Styled with CSS
- Gradient background
- Fade-in animation
- Console output
- Alert popup

---

### 3. DOM Manipulation - `hello-world-dom.js`
JavaScript script demonstrating DOM manipulation techniques.

**Usage:**
Include in an HTML file:
```html
<script src="hello-world-dom.js"></script>
```

**Features:**
- createElement and appendChild
- innerHTML manipulation
- Template literals
- Dynamic styling

---

### 4. Python - `hello-world.py`
Python Hello World with various implementations.

**Run:**
```bash
python3 hello-world.py
```

**Features:**
- Simple print statement
- Function-based approach
- Class implementation
- String formatting
- Loops and list comprehensions

---

### 5. Bash Script - `hello-world.sh`
Shell script with multiple Hello World variations.

**Run:**
```bash
bash hello-world.sh
# or
chmod +x hello-world.sh
./hello-world.sh
```

**Features:**
- Simple echo
- Variables
- Functions
- Loops
- Colored output
- Date integration

---

## Quick Test All Scripts

Run all scripts at once:

```bash
# JavaScript
node hello-world.js

# Python
python3 hello-world.py

# Bash
bash hello-world.sh
```

## Project Context

These Hello World scripts are part of the NovaSphere project, demonstrating basic programming concepts across multiple languages commonly used in web development.
