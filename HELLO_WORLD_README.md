# Hello World Scripts Collection

This directory contains multiple "Hello World" implementations in various languages and formats.

## Available Scripts

### 1. **hello.html** - Standalone HTML Page
A beautiful, animated HTML page with CSS styling.
```bash
# Open in browser or serve with:
python3 -m http.server 8000
# Then visit: http://localhost:8000/hello.html
```

### 2. **hello.js** - Browser JavaScript
JavaScript for browser console and DOM manipulation.
```bash
# Include in HTML:
<script src="hello.js"></script>
```

### 3. **hello-node.js** - Node.js Script
Executable Node.js script with colors and ASCII art.
```bash
node hello-node.js
# Or directly:
./hello-node.js
```

### 4. **hello-server.js** - HTTP Server
Node.js HTTP server serving Hello World page.
```bash
node hello-server.js
# Visit: http://localhost:3000
```

### 5. **hello.py** - Python Script
Python implementation with functions and classes.
```bash
python3 hello.py
# Or directly:
./hello.py
```

### 6. **hello.sh** - Bash Script
Shell script with colored output.
```bash
bash hello.sh
# Or directly:
./hello.sh
```

## Quick Test All Scripts

```bash
# Node.js
node hello-node.js

# Python
python3 hello.py

# Bash
bash hello.sh

# HTTP Server (background)
node hello-server.js &
curl http://localhost:3000
```

- ✅ Multiple language implementations
