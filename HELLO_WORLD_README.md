# Hello World Scripts Collection

This directory contains various "Hello World" implementations in JavaScript for different contexts.

## Available Scripts

### 1. **hello-world.js**
Basic JavaScript hello world with multiple approaches:
- Simple console.log
- Function-based greeting
- Class-based greeter
- Module exports for reusability

**Run:**
```bash
node hello-world.js
```

### 2. **hello-world-node.js**
Node.js specific implementation featuring:
- Process information
- Async/await patterns
- Colored terminal output
- Executable script (with shebang)

**Run:**
```bash
node hello-world-node.js
# or
./hello-world-node.js
```

### 3. **hello-world-es6.js**
Modern ES6+ features demonstration:
- Arrow functions
- Destructuring
- Spread operators
- Template literals
- Promises and async/await
- Classes with static methods
- ES6 modules (export)

**Run:**
```bash
node hello-world-es6.js
```

### 4. **hello-world-dom.js**
DOM manipulation examples:
- Dynamic element creation
- Event listeners
- Inline styling
- Interactive buttons
- Alternative innerHTML approach

**Use:** Include in an HTML file:
```html
<script src="hello-world-dom.js"></script>
```

### 5. **hello-world.html**
Complete standalone HTML page:
- Embedded CSS styling
- Gradient background
- Interactive button
- Responsive design
- Inline JavaScript

**Run:** Open in a web browser or use a local server:
```bash
python3 -m http.server 8000
# Then visit http://localhost:8000/hello-world.html
```

## Quick Test All Scripts

```bash
# Test JavaScript scripts
node hello-world.js
node hello-world-node.js
node hello-world-es6.js

# View HTML in browser
# Open hello-world.html in your browser
```
