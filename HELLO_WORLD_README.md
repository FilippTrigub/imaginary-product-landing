# Hello World Scripts

This directory contains various "Hello World" scripts in different programming languages.

## Available Scripts

### Python
- File: `hello_world.py`
- Run: `python3 hello_world.py`

### JavaScript
- File: `hello_world.js`
- Run: `node hello_world.js`

### Shell Script
- File: `hello_world.sh`
- Run: `./hello_world.sh` (make sure it's executable with `chmod +x hello_world.sh`)

### HTML
- File: `hello_world.html`
- Open in any web browser to see the output.

## About
Each script outputs the classic "Hello, World!" message and follows common conventions for the respective language including basic documentation.
