# Hello World Scripts

This directory contains simple "Hello, World!" programs in various programming languages.

## Available Scripts

### Python (`hello.py`)
```bash
python3 hello.py
# or
./hello.py
```

### JavaScript/Node.js (`hello.js`)
```bash
node hello.js
# or
./hello.js
```

### Bash (`hello.sh`)
```bash
bash hello.sh
# or
./hello.sh
```

### Java (`Hello.java`)
```bash
javac Hello.java
java Hello
```

### C (`hello.c`)
```bash
gcc hello.c -o hello
./hello
```

## Purpose

These scripts demonstrate the basic "Hello, World!" program structure in different programming languages, serving as a starting point for learning each language's syntax.
