# Hello World Scripts

A collection of "Hello, World!" scripts in different programming languages.

## Available Scripts

### JavaScript (Node.js)
```bash
node hello.js
```

### Python
```bash
python3 hello.py
# or make it executable and run directly
./hello.py
```

### Bash
```bash
bash hello.sh
# or make it executable and run directly
./hello.sh
```

### Node.js HTTP Server
```bash
node hello-server.js
# Then visit http://127.0.0.1:3000/ in your browser
```

## Files

- `hello.js` - Simple JavaScript console output
- `hello.py` - Simple Python print statement
- `hello.sh` - Simple Bash echo command
- `hello-server.js` - Node.js HTTP server that responds with "Hello, World!"
