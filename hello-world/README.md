# Hello World Scripts

A collection of "Hello, World!" scripts in various programming languages commonly used in web development.

## Scripts Included

### 1. **hello-world.js** - Browser JavaScript
A simple JavaScript script that outputs "Hello, World!" to the console.

```bash
# Run in browser console or with any JavaScript runtime
node hello-world.js
```

### 2. **hello-world.html** - HTML
A minimal HTML page that displays "Hello, World!" as a heading.

```bash
# Open in a web browser
open hello-world.html
# or
firefox hello-world.html
```

### 3. **hello-world-node.js** - Node.js
A Node.js executable script with shebang for direct execution.

```bash
# Make executable
chmod +x hello-world-node.js

# Run directly
./hello-world-node.js

# Or run with Node.js
node hello-world-node.js
```

### 4. **hello-world.sh** - Bash Shell Script
A simple Bash script that outputs "Hello, World!" using the `echo` command.

```bash
# Make executable
chmod +x hello-world.sh

# Run directly
./hello-world.sh

# Or run with bash
bash hello-world.sh
```

## Quick Start

```bash
# Make all scripts executable
chmod +x *.sh hello-world-node.js

# Run all scripts
node hello-world.js
bash hello-world.sh
./hello-world-node.js

# Open HTML in browser
open hello-world.html
```

## Purpose

These hello-world scripts serve as:
- **Learning examples** for beginners in different programming languages
- **Templates** for starting new projects
- **Testing tools** to verify that your development environment is set up correctly
- **Documentation** of the languages and tools used in this project

## Languages and Technologies

- **JavaScript (ES6)** - Modern JavaScript syntax
- **HTML5** - Semantic markup
- **Node.js** - JavaScript runtime for server-side execution
- **Bash** - Unix shell scripting

All scripts demonstrate the simplest possible program: printing "Hello, World!" to the output.
