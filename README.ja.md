# NovaSphere v2 ランディングページ

NovaSphere v2 は、ホログラフィック・インターフェース、量子プロセッシング、ニューラル・セキュリティを備えた「パーソナルコンピューティングの未来」を紹介するコンセプトのランディングページです。本リポジトリには、マーケティングサイトに必要なマークアップ、スタイル、スクリプト、メディアなどのアセットが含まれています。

## 特長

- **ヒーローセクション**: NovaSphere デバイスを紹介し、行動喚起ボタンを配置
- **機能グリッド**: ホログラフィック・インターフェース、量子プロセッシング、ニューラル・セキュリティ、無限バッテリーを強調
- **価格プラン**: Standard、Pro、Enterprise の3つの階層を用意
- **プロダクトビジョン**: 統計情報、長期ロードマップ、ビジュアルを含むストーリーテリング
- **About セクション**: チームのビジョンを紹介
- **お問い合わせフォーム**: ユーザーからの問い合わせを収集
- **フッター**: ナビゲーション、連絡先情報、法務リンクを掲載

## プロジェクト構成

```
/
├── index.html        # メインのランディングページ
├── header.html       # オプションのヘッダー用マークアップ（デフォルトでは未リンク）
├── team.html         # チームページのプレースホルダー
├── styles.css        # サイト全体のスタイル
├── script.js         # JavaScript プレースホルダー
├── images/
│   ├── logo.svg      # NovaSphere ロゴアセット
│   └── product.svg   # デバイスのイラスト
└── README*.md        # 複数言語のドキュメント
```

## はじめに

1. このリポジトリをクローンまたはダウンロードします。
2. `index.html` を PC またはモバイルの最新ブラウザで開きます。
3. アセットが正しく読み込まれるよう、`images/` ディレクトリを HTML ファイルと相対位置のまま保持してください。

> このページは完全な静的サイトであり、Google Fonts が提供する Montserrat 以外にビルドプロセスや外部依存は必要ありません。

## カスタマイズ方法

- `index.html` のテキストを編集して、製品メッセージを自分用に調整します。
- `styles.css` のカラーパレットやタイポグラフィを変更して、ブランドに合わせます。
- `images/logo.svg` と `images/product.svg` を独自のアセットに差し替え、ファイル名を揃えるか HTML 内の参照パスを更新します。
- `script.js`（現在は空）にロジックを追加して、インタラクションを強化します。

## アクセシビリティとパフォーマンスの注意点

- セマンティックな HTML 構造と明確な見出し・セクション ID によりナビゲーションしやすい構成
- フォームコントロールには対応するラベルを付与
- CSS の Flexbox を用いたレスポンシブレイアウト
- 外部リソースを最小限に抑え、読み込み時間を短縮

## ライセンス

本プロジェクトはデモ目的で提供されています。個人または教育用途での再利用・改変は自由ですが、商用利用の際は既存の商標名を尊重し、ご自身のブランド名に置き換えてください。
