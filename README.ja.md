## 🚀 はじめに

1. このリポジトリをクローンします：
   ```bash
   git clone <リポジトリURL>
   cd vercel/sandbox
   ```

2. Webブラウザで`index.html`を開きます：
   ```bash
   # Pythonの組み込みサーバーを使用
   python3 -m http.server 8000
   
   # またはNode.jsのhttp-serverを使用
   npx http-server -p 8000
   ```

3. ブラウザで`http://localhost:8000`にアクセスします
