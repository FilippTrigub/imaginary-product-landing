## 📖 NovaShpereについて

2024年に設立されたNovaShpereは、エンジニア、デザイナー、思想家を結集し、人間の創造性を拡張するツールを構築しています。私たちの製品は、個人クリエイターから大規模組織まで拡張可能な、楽しく意味のあるインタラクションに焦点を当てています。

NovaSphere v2は単なるデバイスではなく、私たちの働き方、創造方法、つながり方を再構築するためのプラットフォームです。すべての表面、センサー、ピクセルをデザインし、それらが消えることで、あなたのアイデアが主役になれるようにしました。
