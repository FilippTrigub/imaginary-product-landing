# Hello World Scripts

This directory contains various "Hello World" scripts in different programming languages and formats.

## Available Scripts

### 1. Python Script (`hello.py`)
A simple Python hello world script.

**Run with:**
```bash
python3 hello.py
# or
./hello.py
```

### 2. Node.js Script (`hello.js`)
A simple Node.js hello world script.

**Run with:**
```bash
node hello.js
# or
./hello.js
```

### 3. Bash Script (`hello.sh`)
A simple Bash shell hello world script.

**Run with:**
```bash
bash hello.sh
# or
./hello.sh
```

### 4. HTML Page (`hello.html`)
A styled HTML hello world page with animations.

**View by:**
- Opening directly in a web browser
- Or serving with a local server:
  ```bash
  python3 -m http.server 8000
  # Then visit: http://localhost:8000/hello.html
  ```

### 5. HTTP Server (`hello-server.js`)
A Node.js HTTP server that serves a hello world page.

**Run with:**
```bash
node hello-server.js
# Then visit: http://localhost:3000/
```

Press `Ctrl+C` to stop the server.
