# Hello World Scripts

This directory contains Hello World examples in multiple programming languages.

## Available Scripts

### JavaScript (Node.js)
```bash
node hello.js
```

### Python
```bash
python3 hello.py
# or
./hello.py
```

### Bash
```bash
bash hello.sh
# or
./hello.sh
```

### HTML
Open `hello.html` in a web browser to see the styled Hello World message.

### C
```bash
gcc hello.c -o hello_c
./hello_c
```

### C++
```bash
g++ hello.cpp -o hello_cpp
./hello_cpp
```

### Java
```bash
javac hello.java
java hello
```

### Ruby
```bash
ruby hello.rb
# or
./hello.rb
```

### Go
```bash
go run hello.go
```

### Rust
```bash
rustc hello.rs
./hello
```

## Tested Scripts

The following scripts have been tested and verified to work:
- ✅ JavaScript (Node.js 22)
- ✅ Python 3
- ✅ Bash

Other scripts require their respective compilers/interpreters to be installed.
