Hello World Scripts Demo

NovaSphere Project - Multiple Hello World Implementations

Available Scripts

This repository contains several "Hello World" scripts in different languages and formats:

  • hello.js - Basic Node.js console script
  • hello-browser.js - Interactive browser script (loaded on this page!)
  • hello-server.js - Full HTTP server with multiple routes
  • hello.sh - Bash shell script with colored output
  • hello.py - Python script with classes and functions

Usage Examples

Node.js Script

$ node hello.js
Hello, World!
Welcome to NovaSphere!

Shell Script

$ bash hello.sh
# or
$ chmod +x hello.sh && ./hello.sh

Python Script

$ python3 hello.py
# or
$ chmod +x hello.py && ./hello.py

HTTP Server

$ node hello-server.js
# Then visit http://localhost:3000
Note: The HTTP server includes multiple routes:
  • / - Main page with styled interface
  • /api - Plain text API endpoint
  • /json - JSON response with metadata
  • /about - About page

Features

  • Multiple programming languages (JavaScript, Python, Bash)
  • Both CLI and browser implementations
  • Full HTTP server with routing
  • Colored terminal output (shell script)
  • Object-oriented examples (Python)
  • Module exports (Node.js)
  • Graceful shutdown handling (server)
  • Responsive design (browser/server)
