## 🌟 プロジェクト紹介

## 🚀 クイックスタート

### ローカル実行

1. **リポジトリをクローン**
   ```bash
   git clone https://github.com/novasphere/v2.git
   cd novasphere-v2
   ```

2. **ローカルサーバー起動**
   ```bash
   # Pythonを使用（推奨）
   python -m http.server 8000
   
   # またはNode.jsを使用
   npx serve .
   
   # またはPHPを使用
   php -S localhost:8000
   ```

3. **ブラウザでアクセス**
   ブラウザを開き`http://localhost:8000`にアクセス

### 直接アクセス

`index.html`ファイルをブラウザで直接開くことも可能です。
