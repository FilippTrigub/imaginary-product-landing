# Lesson 1: JavaScript Basics

Welcome to your first set of JavaScript exercises! In this lesson, you'll learn about:
- Variables (let, const)
- Data types (strings, numbers, booleans)
- Basic operators (+, -, *, /, %)
- Template literals

## Instructions
1. Open each exercise file
2. Read the instructions in the comments
3. Write your code where indicated
4. Run the file with: `node filename.js`
5. Check your answers against the solution files

## Exercises
1. `exercise-01-variables.js` - Learn about variables
2. `exercise-02-datatypes.js` - Work with different data types
3. `exercise-03-operators.js` - Practice arithmetic operations
4. `exercise-04-strings.js` - String manipulation basics

Good luck!
