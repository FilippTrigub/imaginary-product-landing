# Hello World Scripts

This directory contains simple "Hello, world!" scripts in several languages.

Available scripts:

- `hello.sh` - Bash
- `hello.py` - Python 3
- `hello.js` - Node.js
- `hello.rb` - Ruby
- `hello.go` - Go (compile with `go build` or run with `go run`)

Make scripts executable and run, e.g.:

```bash
chmod +x hello.sh hello.py hello.js hello.rb
./hello.sh
./hello.py
./hello.js
./hello.rb
go run hello.go
```
