# Hello World Scripts

This directory contains simple "Hello, World!" scripts in various programming languages.

## Available Scripts

- **hello.js** - JavaScript (Node.js)
  ```bash
  node hello.js
  # or
  ./hello.js
  ```

- **hello.py** - Python 3
  ```bash
  python3 hello.py
  # or
  ./hello.py
  ```

- **hello.sh** - Bash shell script
  ```bash
  bash hello.sh
  # or
  ./hello.sh
  ```

- **hello.rb** - Ruby
  ```bash
  ruby hello.rb
  # or
  ./hello.rb
  ```

- **hello.pl** - Perl
  ```bash
  perl hello.pl
  # or
  ./hello.pl
  ```

All scripts are executable and can be run directly with the shebang notation (./script_name) or by invoking the interpreter explicitly.
