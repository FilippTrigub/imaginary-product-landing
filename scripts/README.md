# Hello Scripts

This folder contains simple "Hello, world!" scripts in several languages:

- `hello.sh` — Bash
- `hello.py` — Python 3
- `hello.js` — Node.js
- `hello.rb` — Ruby (may not be installed in the environment)

Run executable scripts with `./scripts/<name>` or run with the interpreter, e.g. `node scripts/hello.js`.
