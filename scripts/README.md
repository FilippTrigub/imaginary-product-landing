# Hello World Scripts

This directory contains simple "Hello, World!" scripts in various programming languages.

## Available Scripts

- `hello.js` - JavaScript (Node.js)
- `hello.py` - Python 3
- `hello.sh` - Bash
- `hello.rb` - Ruby
- `hello.pl` - Perl

## Usage

All scripts are executable and can be run directly:

```bash
./hello.js
./hello.py
./hello.sh
./hello.rb
./hello.pl
```

Or with their respective interpreters:

```bash
node hello.js
python3 hello.py
bash hello.sh
ruby hello.rb
perl hello.pl
```

Each script outputs: `Hello, World!`
