## 📖 关于 NovaSphere

NovaSphere 成立于2024年,汇集了工程师、设计师和思想家,共同打造扩展人类创造力的工具。我们的产品专注于令人愉悦、有意义的交互,可以从个人创作者扩展到大型组织。

NovaSphere v2 不仅仅是一台设备——它是一个重新构想我们工作、创作和连接方式的平台。我们设计每一个表面、传感器和像素都是为了让它们消失,让您的想法成为主角。
