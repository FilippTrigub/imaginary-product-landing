## 🚀 Iniziare

### Prerequisiti
- Un browser web moderno (Chrome, Firefox, Safari, Edge)
- Opzionale: Server web locale per lo sviluppo

### Installazione

1. **Clonare il repository**
   ```bash
   git clone <url-repository>
   cd /vercel/sandbox
   ```

2. **Aprire nel browser**
   
   Semplicemente aprire `index.html` nel browser web:
   ```bash
   # Usando il browser predefinito
   open index.html
   
   # O con un browser specifico
   firefox index.html
   chrome index.html
   ```

3. **Opzionale: Eseguire con server locale**
   
   Per una migliore esperienza di sviluppo:
   ```bash
   # Python 3
   python3 -m http.server 8000
   
   # Node.js (con http-server)
   npx http-server -p 8000
   ```
   
   Quindi visitare `http://localhost:8000`
